## Description

To finish this you will need 1 x skate bearing, 8mm bolt 30mm long, 2 x 8mm washers and an 8mm nut.
You can find Peter's jig here https://www.thingiverse.com/thing:5941354
and a video about this machine here: https://youtu.be/zfD8-2_CmqM
 It can either be exported as svg and laser cut in 5mm acrylic or 3D printed
