## Description

This is the generator section to go with the rubber band motor which can be found here

you can find a video about this here https://youtu.be/6lrnD_-gFsY

and to finish it you will need 2 x bearings 22mm by 8mm by 7mm, 36 circular neodymium  magnets 10mm by 1mm an some magnet wire in the range of 0.2mm to 0.05mm - any is good but the thin stuff is hard to handle - the thinner the wire the more turns you can get in the space so the higher the voltage - wind a serpentine coil with as much wire as will fit in the cassette

you will also need to build the rubber band motor and the files for that can be found here: https://www.thingiverse.com/thing:6784387
