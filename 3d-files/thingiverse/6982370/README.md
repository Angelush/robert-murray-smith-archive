## Description

You can find a video about this here https://youtu.be/zrVggxqgHX0
To finish the linear bearing you will need 2 x 12mm OD 6mm ID 4mm thick bearings.
To finish the planetary version you will need 6 x 19mm OD 8mmID 6mm thick
The planetary version uses the same TPU band and the same small rollers as the rotary version
