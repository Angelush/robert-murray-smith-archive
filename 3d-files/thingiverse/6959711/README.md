## Description

You can find a vdeo about this here https://youtu.be/z_FdCGIzG-w

This is mostly printed in PLA but the rollers are TPU 95A. I addition you will need 1 x thrust bearing 42mm by 25mm by 11mm - if you can't get that change the size of the recess as it is really not that critical - you could probably do without it if you wanted. The rollers are printed 100% infil
