## Description

To finish this you will also need 12 x 19mm OD by 8mm ID by 6mm thick bearings for the rollers

You can find a video about this here https://youtu.be/SPo5xVkJDAs

All TPU inserts are printed with zero wall thickness for top and base, wall thickness is 2 and infill is 10% triangular
