## Description

This has 96 parts and you will also need acrylic rod - 8mm by 600mm, 6mm by 300mm and 4mm by 60mm for the axles

There is a logic - part 1 is the base, Part 2 is the outer 3 planets, Part 3 is the inner 3 planets, part 4 is the case and drive is - as it says the drive mechanism.

But, in order to have a chance at putting this togeter you need to see the build video here https://youtu.be/7aHL_ME6vwc

There is another video talking about to set it up here https://youtu.be/Q-HfTxC-0Iw

This is step by step on how to construct it. These are all STL files so to laser cut open the file in something like Tinkercad and export as SVG.

The drive was adapted from an OT Vinta gear and I didn't design it from scratch, it was heavily based on this model but converted into a model that is able to be laser cut and 
printed: https://grabcad.com/library/antikythera-mechanism-4
