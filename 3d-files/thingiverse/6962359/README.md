## Description

To finish this you will need 4 x 22mm by 8mm by 7mm bearings and 2 x 19mm by 10mm by 4mm bearings. The roller cover is in 95A TPU and the support sides have small dots on them - when making the cage make sure both dots are on the same side and face each other. To retain the roller there are two clips that fit on the roller axle
