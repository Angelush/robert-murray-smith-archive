## Description

You can find  video about this here: https://youtu.be/BANZ_x-4GgY

In order to make this you will also need 
2 x 22mm by 8mm by 7mm bearings
2 x 12mm by 8mm by 4mm bearings
an M10 bolt 70mm long
some magnet wire about 0.2mm
A relay
12 x 15mm by 5mm neodymium magnet
glue with cyano acrylic glue
