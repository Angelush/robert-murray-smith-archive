## Description

This is a remix of https://www.thingiverse.com/thing:145186 to enable it to be cut on a laser cutter and because i found the gear teeth size too flimsy for this job so i resized them.

you will also need 50cm 8mm acrylic round bar, 8 x 608 bearings (Skate bearings) 4 x 12m by 8mm by 3mm bearings, 24 x 15mm by 5mm round neofymium magnets, some 0.1mm magnet wire

It can be 3d printed or laser cut - to 3d print down load the files and off you go to laser cut import the files into tinkercad and re-export as SVG files - the parts are colour coded - cream is 10mm, orange is 5mm and blue is 2mm thick acrylic.

you can find a video on the build here https://youtu.be/_OiqbNHWUqo
