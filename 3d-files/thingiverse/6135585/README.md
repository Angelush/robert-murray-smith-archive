## Description

An unusual arrangement in that it uses magnetic bearings - for the bearings you will need 1 12mm ball bearing,2 x 15mm by 3mm neodymium magnets, 1 x 10mm by 5mm magnet nd 8 x 10mm x 2mm magnets
