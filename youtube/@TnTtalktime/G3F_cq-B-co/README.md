## Overview

Robert shares the "Parable of the Chimpanzees" — a story about how learned behaviours persist long after the original reasons for them have vanished. He uses it to argue that conventional wisdom around testing generators (using a resistive load) is not the only valid approach, and that charging a capacitor can reveal the same information.
