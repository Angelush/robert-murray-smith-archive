## Overview

Robert Murray-Smith explores the concept of control in science, using the evolution of pig farming as a central metaphor. He argues that traditional science — like intensive farming — has been defined by a desire to forcibly control nature, which is both brutal and inefficient. The video's key takeaway is that a shift is underway toward *working with* nature rather than against it, yielding better outcomes with far less energy.
