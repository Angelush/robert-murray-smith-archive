## Overview

Robert demonstrates how a motor can function as a generator, showing that voltage output depends on magnet strength, number of wire turns, and spin speed. Using a gear system with a 10:1 ratio (120-tooth wheel driving a 12-tooth wheel), he converts slow human cranking into fast enough rotation to produce 36 volts — enough to light an LED panel. The core lesson is that gearing up speed always demands a proportional increase in torque at the input, and every power system hits a hard limit defined by its input force.
