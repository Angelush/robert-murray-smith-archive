## Overview

Robert responds to a viewer question about how to get started with making and experimenting. His core message is deceptively simple: the best way to start is simply to start. He argues that over-preparation, procrastination, and fixating on the end product are the main obstacles to learning, and that genuine expertise comes through doing.
