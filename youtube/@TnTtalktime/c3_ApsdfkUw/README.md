## Overview

Robert demonstrates how particle size — specifically the surface area to volume ratio — dramatically affects the speed and intensity of oxidation reactions. Using steel wool, iron powder, and thermite, he shows that the same material (iron) can rust imperceptibly slowly, burn visibly, or react with explosive violence depending purely on how finely divided it is. The key takeaway is that smaller particles expose more surface for reactions, which is also why nanomaterials are so significant.
