## Overview

Robert Murray-Smith argues against rigid, binary thinking in science — particularly the habit of asking "what's the best X?" without context. Using passive electronic components and energy storage devices as examples, he demonstrates that real-world devices always contain a mixture of effects (capacitive, resistive, inductive, battery-like), and that what we *call* something is simply determined by which effect predominates. The core takeaway is that context determines the correct answer, and that demanding a single "truth" leads to meaningless conclusions.
