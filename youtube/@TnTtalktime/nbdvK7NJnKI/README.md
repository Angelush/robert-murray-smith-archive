## Overview

Robert Murray-Smith demystifies solid state batteries, explaining that they differ from conventional batteries only in having a solid rather than liquid electrolyte. He argues that the real barrier is not technological — solid state batteries have existed since the 1970s — but economic: the question the public is actually asking is when affordable solid state batteries will arrive, not merely when they will exist.
