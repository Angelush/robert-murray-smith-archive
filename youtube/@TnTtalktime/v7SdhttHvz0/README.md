## Overview

Robert Murray-Smith uses this short video to gently encourage his community to think problems through before posting comments — specifically, to consider whether a problem is real and, if so, to propose a solution alongside it. Using his maglev wind turbine project as a concrete example, he demonstrates that several commonly raised concerns have simple, obvious fixes, and argues that solution-oriented comments add far more value to the community than problem-only observations.
