## Overview

This video explores Vertical Axis Wind Turbines (VAWTs) as an alternative to conventional Horizontal Axis Wind Turbines (HAWTs), and investigates how biomimicry — specifically humpback whale tubercles and owl wing trailing-edge serrations — can improve their performance. Robert 3D prints prototype VAWT blades with and without these biological features and measures the RPM difference using a digital tachometer, demonstrating meaningful gains including self-starting capability.
