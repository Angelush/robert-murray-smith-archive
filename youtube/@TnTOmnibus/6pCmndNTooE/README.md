## Overview

Robert Murray-Smith builds an unusual Wimshurst electrostatic machine using large circular glass display shelves as the rotating discs instead of the typical plastic. The key innovation is a hubless, axle-free cradle design that uses rim-mounted pulley wheels and motors to drive the discs, allowing the machine to be scaled to virtually any disc size. The completed machine successfully generates high-voltage sparks.
