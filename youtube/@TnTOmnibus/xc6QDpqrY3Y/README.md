## Overview

Robert Murray-Smith builds a "Wind Wall" — a vertical-axis wind turbine array inspired by an online design he later discovered was only a render and had never actually been built. After his initial flat-blade version fails to turn in the wind, he iterates to a Savonius-type rotor design using uPVC guttering, which works successfully and generates electricity via repurposed NEMA stepper motors acting as generators.
