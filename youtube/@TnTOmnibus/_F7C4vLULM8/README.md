## Overview

Robert Murray-Smith dismantles an expired CO2 fire extinguisher to explore its reusable components, primarily the thick aluminum tube that forms the body. He demonstrates three distinct projects using salvaged parts: building a resonant bell-speaker using a piezo element, turning the aluminum ring into DIY headphones, and constructing a simple beta/gamma-type Stirling engine using the tube as the cylinder.
