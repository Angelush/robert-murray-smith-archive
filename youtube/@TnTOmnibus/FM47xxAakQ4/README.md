## Overview

Robert Murray-Smith demonstrates how to build an emergency pumping and power generation system entirely from widely available materials. He constructs ball check valves from plumbing fittings and ping pong balls, assembles accordion (concertina) bellows from venetian blind slats and tape, combines them into a working water pump, and then drives a 3D-printed vane motor to generate electricity. The central takeaway is that this modular system — pump, valves, and generator — can be built cheaply and without power tools from scavenged materials in an emergency.
