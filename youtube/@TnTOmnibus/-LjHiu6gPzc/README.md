## Overview

Robert Murray-Smith explores electrostatic generation as an alternative to traditional coil-and-magnet generators, replicating a Hackaday project that uses a variable capacitor to harvest energy. He builds a Bennett doubler-style electrostatic generator using acrylic discs and aluminium foil, then replaces the metal plates with his homemade graphite-based conductive ink — achieving comparable or better results, and demonstrating that triboelectric pre-charging significantly boosts output.
