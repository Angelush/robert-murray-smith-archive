## Overview

Robert Murray-Smith uses the design and build of a folding portable wind turbine as a practical framework for teaching how all electrical power generation systems work. He breaks down any generation system into three universal components — input, link (gearing), and output (generator) — and walks through building each element: a serpentine-coil axial-flux generator, a 3-stage compound planetary gearbox, and folding turbine blades cut from PVC drain pipe. The key takeaway is that "best" design choices are always contextual; understanding the three-component framework lets you reason through any power generation scenario.
