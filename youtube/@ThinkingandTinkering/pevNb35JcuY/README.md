## Overview

Robert Murray-Smith provides a step-by-step tutorial on building a Tesla turbine configured as an air-powered generator, using everyday materials centered around DVD discs. He walks through each fabrication stage from cutting the acrylic drive plate to assembling the disc pack, adding generator components (magnets, ferrite rods, and coils), and housing everything in a repurposed DVD spindle case. The video concludes with a live spin test using a hand pump to demonstrate the device in motion.
