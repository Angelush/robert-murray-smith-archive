## Overview

Robert Murray-Smith demonstrates how to synthesize graphitic carbon nitride (g-C3N4) from cheap, readily available urea as a low-cost photocatalytic material for splitting water into hydrogen using sunlight. He then shows how to improve its photocatalytic performance by doping it with iron(III) chloride, producing a material that would cost ~£900 commercially for just a few pounds in materials and electricity.
