## Overview
This video, presented by Luke, is a follow-up to an earlier laser engraving video, responding to viewer suggestions. The goal is to create a clean workshop plaque on pine wood by removing image backgrounds and using simple vector outlines, resulting in a professional-looking engraved sign. The key improvement over the previous video is stripping backgrounds from images so the laser only engraves meaningful outlines.

## Key Topics
- Designing a workshop plaque for display in a maker/workshop space
- Removing image backgrounds to improve laser engraving clarity
- Using white-filled shapes to prevent the laser from cutting along design edges
- Importing a design into laser engraver control software (VigoWorks)
- Dialing in laser intensity and speed settings for pine wood

## Materials and Chemicals Mentioned
- Pine wood (plaque substrate)

## Techniques and Methods
- Vector design creation: rectangle, text box, and image layout (hammer and screwdriver outlines) grouped and exported as JPEG
- Setting rectangle and border color to white to suppress unwanted laser cutting on edges
- Importing design via USB into VigoWorks software
- Laser engraving at **25% intensity** and **460 mm/min** speed, design sized **290 mm × 50 mm**
