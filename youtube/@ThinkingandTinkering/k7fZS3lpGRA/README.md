## Overview

This follow-up teardown video walks through the internal components of a homemade magnetic hot plate stirrer built using conductive ink, copper tape, and basic materials. The presenter disassembles the device to explain how each of the four main components works, highlighting that the resistive heater achieves approximately 99% efficiency converting electricity to heat. The build demonstrates that functional lab equipment can be constructed primarily from acrylic, aluminium sheet, epoxy resin, and conductive ink.
