## Overview

Robert Murray-Smith demonstrates how to make a casein-based bioplastic that behaves like rubber, using milk protein (casein), potassium hydroxide, and sulfur. The resulting material is flexible, waterproof, and can be used to coat fabrics — similar to a Macintosh raincoat. He extends the experiment by adding carbon to create a flexible conductive ink with potential uses in wearable heating elements and electronics.
