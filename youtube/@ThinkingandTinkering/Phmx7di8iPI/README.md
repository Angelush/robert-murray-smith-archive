## Overview

Robert Murray-Smith reviews the Vevor tent stove (a budget-friendly ~£150 outdoor wood stove) and demonstrates how to convert it into a safe indoor heater by using a homemade two-tin alcohol burner as the fuel source instead of wood. The key takeaway is that any enclosed stove can serve as an emergency indoor heater when paired with an ethanol/methanol burner, since those fuels don't require venting, making the setup compliant with UK legislation for unvented indoor use.
