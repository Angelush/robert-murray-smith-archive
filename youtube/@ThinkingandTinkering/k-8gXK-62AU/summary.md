## Overview
This video, presented by Luke, demonstrates how to recover deleted or accidentally formatted files from USB drives, SD cards, and hard drives using the free software Recuva (made by the creators of CCleaner). The tutorial walks through a live example of formatting a USB stick and successfully recovering all five files using Recuva's deep scan feature. The main takeaway is that file recovery is straightforward and often fully successful when done promptly with the right tool.

## Key Topics
- Introduction to Recuva as a free file recovery tool
- Demonstrating intentional formatting of a USB stick to simulate data loss
- Step-by-step walkthrough of the Recuva recovery process
- Using the deep scan option for more thorough file detection
- Best practice of recovering files to a different location (e.g., desktop) rather than back to the original drive
- Cross-platform caveats: files deleted on a Mac and recovered on Windows may be unrecoverable (indicated by a red dot in Recuva)
- Uncertainty around recovering files deleted on one Windows machine and recovered on another

## Materials and Chemicals Mentioned
None mentioned.

## Techniques and Methods
- Full format of a USB drive to simulate data loss
- Running Recuva's standard scan followed by optional deep scan
- Browsing to a specific drive location within Recuva rather than scanning all drives
- Recovering files to an intermediate desktop folder rather than directly back to the source drive
- Visual verification of recovered files (opening images and scrubbing through video files)
