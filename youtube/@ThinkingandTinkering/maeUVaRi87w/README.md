## Overview

This video is part of a series on building a Van de Graaff generator, focusing specifically on wiring the motor to the mains supply. Robert walks through the components required and demonstrates the step-by-step wiring process using UK electrical conventions, connecting a double pole switch, a dimmer switch, and a starting capacitor salvaged from the original donor machine.
