## Overview

Robert Murray-Smith demonstrates how to build a simple air engine using two sets of bellows connected to a handmade rotary valve chest machined from acrylic. The engine requires minimal tools and loose tolerances, making it highly accessible to home builders. While not a powerhouse, the project showcases the versatility of bellows as a construction platform and introduces a clever valve design adaptable to compressed air, steam, vacuum, and Stirling engine applications.
