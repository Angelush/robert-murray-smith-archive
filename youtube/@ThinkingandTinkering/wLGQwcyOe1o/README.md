## Overview

Robert Murray-Smith demonstrates how to design and 3D print a functional steampunk-style pencil sharpener inspired by the Victorian-era "Gem" pencil sharpener patent. The video covers how to model spur gears and bevel gears in TinkerCad, then walks through assembling the printed parts with bearings, sandpaper, and glue into a fully working sharpener. The result is a deliberately over-engineered, durable mechanical device that Robert presents as both a practical tool and a gift-worthy object.
