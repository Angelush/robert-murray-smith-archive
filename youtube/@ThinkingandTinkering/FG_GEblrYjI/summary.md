## Overview
Robert Murray-Smith provides a beginner's first-steps tutorial for OnShape, a free browser-based CAD program. He walks through account setup, workspace navigation, and the fundamental sketch-then-extrude workflow to create a 3D solid object with a hole through it. The video positions OnShape as a natural progression from Tinker CAD, where users graduate from assembling pre-made primitives to defining geometry from scratch.

## Key Topics
- Creating and signing into a free OnShape account
- Navigating the OnShape workspace (planes, origin, view controls)
- Creating and configuring a 2D sketch (rectangle and circle)
- Setting dimensions using millimeters and inches
- Extruding a 2D sketch into a 3D solid
- Selectively extruding regions to create holes
- Comparison of OnShape vs. Tinker CAD design philosophy

None mentioned.

## Techniques and Methods
- **Sketch creation** — drawing 2D shapes (center-point rectangle, center-point circle) on a selected plane
- **Dimensioning** — using the dimension tool to set precise measurements in inches or millimeters with automatic unit conversion
- **Extrusion (Blind type)** — extruding a 2D sketch region to a specified depth to create a 3D solid
- **Selective region extrusion** — choosing which sketch regions to include or exclude (e.g., omitting the circular region to produce a through-hole)
- **View manipulation** — zooming with scroll wheel, rotating with right-click drag, resetting to isometric view, and using "view normal to sketch space"
