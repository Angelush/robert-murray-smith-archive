## Overview

Robert Murray-Smith unboxes and assembles a generic Chinese-made pedal tricycle (purchased for approximately £220) as the first step in a project to convert it into an electric tricycle for under £300. The video responds to viewer feedback that not everyone can build vehicles from scratch using bamboo or rolled newspaper, so this series demonstrates electrifying a production trike instead. Part 1 covers only the mechanical assembly; the electrification is promised for the next video.
