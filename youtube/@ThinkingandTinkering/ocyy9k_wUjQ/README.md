## Overview

Robert demonstrates a hand-made Hall effect sensor constructed from paper painted with graphene ink, showing a live voltage change when a magnet is brought near. He explains the physics behind the Hall effect — how a magnetic field deflects electron flow in a conductor, creating a measurable voltage difference perpendicular to the current. He concludes that graphene ink can serve as a viable material for painting functional Hall effect sensors, opening up possibilities for DIY sensor fabrication.
