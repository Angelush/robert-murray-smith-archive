## Overview

Robert Murray-Smith builds a hand-cranked mechanical power generator using salvaged bicycle parts, timber, and electrical components. By connecting a repurposed 12V tape recorder motor to a gear system driven by a hand crank, he demonstrates that mechanical motion can generate enough electricity to light LEDs. The project shows how common scrap materials can be combined into a functional generator.
