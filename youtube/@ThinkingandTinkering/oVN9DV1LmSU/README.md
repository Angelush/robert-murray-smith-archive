## Overview

Robert Murray-Smith demonstrates an improved method for synthesising copper nanoparticles using two approaches: an aqueous ascorbic acid reduction and a polyol process using diethylene glycol. The resulting ~4 nm PVP-capped particles are stable in suspension and small enough to pass through inkjet nozzles, making them suitable for printing conductive traces that can be sintered with a camera flash.
