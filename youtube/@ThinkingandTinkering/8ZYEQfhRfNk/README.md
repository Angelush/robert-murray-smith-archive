## Overview

A short, playful experiment in which Robert tests whether a small computer fan repurposed as a generator can produce electricity from household water flow. He first tries a low-flow toilet flush, which proves too weak to spin the fan, then moves to a laundry sink tap, which succeeds in lighting an LED.
