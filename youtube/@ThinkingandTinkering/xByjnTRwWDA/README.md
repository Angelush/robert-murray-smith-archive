## Overview

Robert Murray-Smith offers practical advice on Non-Disclosure Agreements (NDAs), warning inventors and collaborators to read them carefully before signing. His main takeaway is that NDAs are two-way agreements, and an overly long or one-sided NDA is a red flag — both about the document itself and the character of the party proposing it.
