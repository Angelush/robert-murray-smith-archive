## Overview

Robert Murray-Smith demonstrates how to build a simple desktop catapult as a Christmas gift project, using cheap cutting boards as a hardwood source. The build requires only four pieces of wood, two dowels, and a couple of elastic bands, with most of the effort going into sanding and finishing rather than complex construction.
