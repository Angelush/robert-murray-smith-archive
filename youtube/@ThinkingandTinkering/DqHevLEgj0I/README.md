## Overview

Robert demonstrates how to quickly design and 3D print a custom gear using parametric CAD software, after a commercial gear keeps slipping. By searching for a gear primitive in the software, adjusting tooth count and size parameters, and slicing the result, he produces a working custom gear in minutes. The takeaway is that 3D printing makes fabricating precision mechanical parts like gears surprisingly fast and easy.
