## Overview

In this short clarification video, Robert addresses recurring confusion about his theory of gravity. He argues that the question "does gravity cause motion, or does motion cause gravity?" is a false dichotomy — a chicken-and-egg problem — because gravity *is* motion. His core claim is that gravity should be understood as the act of a system transitioning from one state to another, regardless of scale.
