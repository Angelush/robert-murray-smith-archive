## Overview

Robert Murray-Smith converts a discarded plywood cable drum into a vertical axis wind turbine (VAWT) using a squirrel-cage rotor design, mounted on a repurposed exercise bike frame. He is refreshingly candid that he does not know if it will work — the point is simply to try. The turbine successfully spins in winds of just 8–10 mph, validating the concept and setting up the next phase of attaching it to a generator.
