## Overview

Robert Murray-Smith demonstrates a 3D-printed mechanical device based on the Huygens clock mechanism that acts as a "mechanical supercapacitor" — smoothing out variable power input (such as wind gusts) before it reaches a generator. The device decouples input from output using a chain-and-weight system, allowing energy to be temporarily stored in raised weights and released steadily, analogous to how an electronic supercapacitor buffers electrical energy. The concept was developed collaboratively with community members Corwin White Horse and Vibe 77 guy.
