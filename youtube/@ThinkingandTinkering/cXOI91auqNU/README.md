## Overview

Robert Murray-Smith builds a working supercapacitor using egg whisks as electrodes, inspired by a suggestion from community member "xyloscope." Activated carbon felt is stuffed inside stainless steel wool unwound from the whisk, submerged in a saltwater electrolyte, and charged at 1.2 V. The device successfully stores and releases energy — demonstrated by running a small motor — proving the concept works even if efficiency is limited.
