## Overview

Robert builds a small wind turbine he calls "the Daisy" using 40 polycarbonate teaspoons glued around acrylic discs, with the blades heat-bent to roughly 30° pitch. Neodymium magnets glued to the back of the rotor turn it into a generator when paired with a salvaged microwave oven turntable coil. The turbine successfully lights a 12V LED when spun by a hair dryer, with Robert noting it outputs about the same as any comparable turbine.
