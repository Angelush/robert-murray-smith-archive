## Overview

Robert completes his steampunk PC build by constructing a fake cathode ray tube (CRT) display using a flat panel LCD monitor dressed up with salvaged CRT components. He builds a decorative CRT shell from cardboard, vacuum cleaner hose, and resin, then finishes it with a conductive metallic ink for a polished steampunk aesthetic. The completed all-in-one "Victorian PC" is functional and intended to become his video editing machine.
