## Overview
A beginner-friendly introduction to programming the Arduino, aimed at overcoming the fear and perceived complexity of getting started. Robert demonstrates how to wire up a push button (micro switch) and use a pre-written sketch from the Arduino Hub to trigger an LED, emphasising that the first steps are simpler than most people expect.

## Key Topics
- Addressing common fears and barriers to starting with Arduino
- Arduino hardware overview and shields
- Recommended starter kit (Elegu/Elegoo ~£25)
- Using the Arduino Hub (Create Hub) as a learning resource
- Finding and using community-written sketches (GNU/public domain)
- Wiring a push button circuit following a hub circuit diagram
- Modifying a sketch minimally to change behaviour

## Materials and Chemicals Mentioned
- Arduino board with shield
- Micro switch (used instead of a standard push button)
- LED
- Resistors (included in starter kit)
- Jumper cables
- Elegu/Elegoo starter kit (plug-in modules and assorted components)

## Techniques and Methods
- Searching the Arduino Hub for pre-written sketches (e.g. "push button arduino" Google search)
- Reading a circuit diagram and replicating the wiring on a physical Arduino board
- Copy-pasting a community sketch into the Arduino IDE and uploading it
- Minimal code modification: replacing the "light LED" action with a custom action
