## Overview

This video demonstrates how to build a DIY power resistor from scratch as a cost-saving alternative to commercial units. The presenter constructs a 35-ohm resistor using kanthal wire wound around a mica core, encased in plaster of Paris. The finished resistor measures 35.1 ohms, closely matching the target resistance.
