## Overview

This video demonstrates how to successfully drill through ceramic without breaking it, a technique the presenter learned after breaking four pieces. The core lesson is that using the right drill bit, applying water for cooling, and letting the drill do the work (no downward pressure) are the keys to clean results. The method is shown step-by-step using progressively larger masonry bits to achieve a final 8mm hole.
