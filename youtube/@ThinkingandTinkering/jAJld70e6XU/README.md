## Overview

Robert Murray-Smith responds to a challenge: design a wind turbine generator for harsh conditions using minimal tools and scrap materials. He builds a Linear Switched Reluctance Generator from salvaged parts — pipe, bolts, aluminium strip, and speaker magnets — demonstrating that useful electricity can be generated without precision machinery. The video concludes with a live measurement showing 0.6V and 40mA generated purely by hand-turning the crank.
