## Overview

This video is a short tutorial on how to import MP4 files into Audacity, the free open-source audio editor, by installing the FFmpeg library. The presenter demonstrates the error that occurs without the library (on Mac), then walks through the Windows installation process step by step. The key takeaway is that installing FFmpeg enables simple drag-and-drop MP4 import directly into Audacity for audio extraction.
