## Overview

A competition entry for Robert Murray-Smith's "Made From Paper" competition, submitted by a non-native English speaker. The video presents a passive, paper-based mechanical hygroscopic sensor that controls a drip irrigation system — watering plants only when the soil is dry, without any electronics. The design addresses key flaws in existing ceramic cone irrigators (clogging, algae, and over-watering) by using a salted-paper element that swells with humidity to pinch shut an air-supply tube, cutting off water flow when the soil is moist.
