## Overview

Robert Murray-Smith demonstrates a fully printed Joule Thief circuit made using graphene ink, building on a previous video about a printed induction coil. The circuit uses inductive coupling through a paper substrate to power an LED, with the two coils remaining electrically isolated from each other. He presents this as a proof-of-concept and teases upcoming step-by-step build videos.
