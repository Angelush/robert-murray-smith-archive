## Overview

Robert demonstrates a 3D-printed universal planetary gearbox designed to interface with a modified brushless DC (BLDC) generator previously built from a broken drill (covered in video 2008). The gearbox is modular and stackable, achieving up to 27:1 gear multiplication across three stages, and is shown generating approximately 17 volts when cranked by hand through an additional adapter gear, proving the concept works.
