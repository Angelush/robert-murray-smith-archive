## Overview

Robert salvages a discarded tower fan from a neighbour's rubbish, extracts its squirrel cage rotor, and mounts a 775 DC motor onto it to create a simple vertical axis wind generator. The build is completed in roughly an hour using minimal tools and off-the-shelf parts, demonstrating that useful wind generation hardware can be assembled from rescued consumer electronics. Robert's primary goal is to show that the concept is approachable and accessible — a proof-of-concept to inspire others to experiment.
