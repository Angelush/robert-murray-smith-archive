## Overview

Robert Murray-Smith upcycles a second-hand pine desk (purchased for £20 from a charity shop) into a handmade wooden doll's house as a Christmas gift for his granddaughter. The finished toy is modular and collapsible, packing back into its own base box, and was built as a deliberate alternative to commercial doll's houses costing £350–£500. Beyond the build itself, Robert uses the project to make a broader point: practicing craft skills on any project — however humble — is essential preparation for more ambitious making endeavors.
