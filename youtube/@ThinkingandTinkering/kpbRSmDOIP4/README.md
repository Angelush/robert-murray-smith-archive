## Overview

Robert Murray-Smith demonstrates a lever-based continuously variable transmission (CVT) built from a scotch yolk mechanism, originally introduced in video 2368. The device uses variable-leverage levers driven by an eccentric pin to produce adjustable rotational output, rectified by ratchets. Crucially, it doubles as a clutch — setting the pin to center position produces zero output while the input continues to spin.
