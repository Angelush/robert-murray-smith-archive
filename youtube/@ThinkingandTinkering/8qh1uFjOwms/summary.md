## Overview
Robert demonstrates how to set up the Qidi X Max 3 3D printer with Cura slicer on Linux, walking through the process of downloading Qidi's official Cura profiles and installing them. The video concludes with a successful test print of parts for a Turgo water turbine project, showcasing the printer's large build volume.

## Key Topics
- Transitioning from Windows to Linux for 3D printing workflows
- Installing Cura on Linux via package manager
- Downloading Qidi X Max 3 slicer profiles from Qidi's website
- Installing printer definition files into Cura's configuration folder
- Adding the Qidi X Max 3 as a printer in Cura
- G-Code adjustments needed to get printing working
- Printing large parts for a Turgo water turbine project

## Techniques and Methods
- Installing Cura on Linux via the distribution's package manager
- Downloading Qidi Cura profiles from Qidi's official website (Software & Firmware section)
- Extracting and copying printer definition files into Cura's configuration folder (`Help > Show Configuration Folder`)
- Restarting Cura to load new printer profiles
- Adding a custom G-Code snippet to machine settings to resolve initial print failure
- Slicing and printing large multi-part assemblies on the Qidi X Max 3
