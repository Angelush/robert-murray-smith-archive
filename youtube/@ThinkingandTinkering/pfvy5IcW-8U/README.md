## Overview

Robert Murray-Smith demonstrates a 3D-printed model of the Archimedes Drive — a friction-based drive system developed by IM Systems — explaining how it works as a toothless planetary gear set using compressible hollow rollers. He builds the model using TPU-printed rollers (mimicking the hollow, springy properties of ping pong balls from his earlier NF friction drive work) and concludes that while plastic home-printed versions suffer from tolerance issues and wear, the concept offers no-backlash torque reduction/increase with great commercial potential in metal production versions.
