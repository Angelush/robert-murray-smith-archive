## Overview

Robert Murray-Smith gives a practical introduction to brushless DC motors, explaining how they work as both motors and generators. He covers the two wiring configurations (Y and delta), demonstrates running salvaged motors using a cheap electronic speed controller, and shows how to build a simple three-phase rectifier to use the same motors as generators. The key takeaway is that any brushless DC motor can be used either as a motor or generator with minimal, inexpensive components.
