## Overview

Robert Murray-Smith demonstrates how to build a DIY marine alcohol stove from cheap cake tins and kitchen items purchased for around £8 total — a fraction of the $300+ price of commercial marine stoves. He walks through the full fabrication process using basic metalworking tools, producing a functional stove that burns methylated spirits or ethanol. The video also introduces a follow-up challenge: adapting the same stove design to run on waste cooking oil for use in a tent.
