## Overview

This video is a competition entry submission for a challenge hosted by Robert Murray-Smith, created by "Ross from Intelligent Tinkering." Ross demonstrates a DIY wind turbine system built from a repurposed fan motor, custom-made blades, and a homemade three-phase bridge rectifier, all wired to a charge controller and battery. The system keeps an LED lit at all times — either powered by live wind generation or by the stored battery when wind stops.
