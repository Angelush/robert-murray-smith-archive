## Overview

Robert Murray-Smith demonstrates a 3D-printed mechanism using three identical bevel gears that achieves a 2:1 gear ratio — directly challenging the conventional wisdom that same-sized gears can only produce a 1:1 ratio. This is the second video in a series on "impossible gears," improving on a previous linkage-based design by using only gears. The assembled device is back-drivable and he suggests it hints at something deeper and important hidden in the mechanism.
