## Overview

Robert Murray-Smith extracts chlorophyll from leaves using isopropyl alcohol and tests whether coating a solar cell with it improves output, exploiting chlorophyll's property of absorbing UV light and re-emitting it as red light — a wavelength solar cells can use more efficiently. The experiment demonstrates approximately a 6% voltage improvement (6.4V to 6.8V), though he notes practical limitations including solvent evaporation and chlorophyll degradation over time.
