## Overview

This video, presented by Luke (not Robert), demonstrates how to recycle old bicycle inner tubes into a functional slingshot. The build combines metalwork, woodworking, and rubber cutting, using almost entirely reclaimed or low-cost materials. The finished slingshot proves to be quite powerful in outdoor testing, with the potential to be made even stronger by layering the rubber.
