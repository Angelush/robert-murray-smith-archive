## Overview

Robert Murray-Smith explores the philosophy of machine design using the concept that complexity is built from simpler subsystems. Using a coil winder as a practical example, he focuses on the challenge of building a mechanical reversing gear mechanism, ultimately reverse-engineering and 3D printing an elegant gear-flipper design by a creator called "Gazumba" (or similar spelling from the transcript).
