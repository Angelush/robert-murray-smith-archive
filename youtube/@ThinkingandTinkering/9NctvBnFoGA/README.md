## Overview

Robert Murray-Smith walks through the basics of building a touch-sensitive lamp circuit, demonstrating how to assemble a lamp base using MDF and a bayonet batten holder, wire it up to a touch switch circuit built around a 555 timer and NPN transistor, and connect a conductive ink touch pad. A key finding demonstrated in the video is that the conductive ink touch pad continues to function correctly even after being over-painted with ordinary emulsion paint.
