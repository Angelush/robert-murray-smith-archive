## Overview

Robert Murray-Smith converts an old electric bench grinder to run on pedal power using his "Dinopod" bicycle-powered attachment, then uses it to sharpen blunt drill bits. The video demonstrates a practical way to repurpose equipment with burnt-out or unusable motors, and doubles as a light argument for human-powered alternatives in contexts where stable electricity isn't available.
