## Overview

Robert Murray-Smith retrofits one of his previously-built rocket stoves with a homemade biodiesel/vegetable-oil burner insert, using a copper preheater coil and a drilled end-cap jet nozzle. He successfully achieves a roaring flame on the first run using vegetable oil, demonstrating proof-of-concept for an oil-burning rocket stove heater. The video serves as a prototype demonstration, with Robert noting several improvements to pursue in future iterations.
