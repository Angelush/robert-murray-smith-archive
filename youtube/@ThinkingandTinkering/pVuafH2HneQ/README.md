## Overview

Robert Murray-Smith demonstrates how to build a large DIY tumble cleaner from a 30-litre food storage container, designed to clean rust, oil, paint, and grease from chunky steel parts. Motivated by the complexity of his earlier shaker-based cleaner (which required welding and a large motor), he devised this simpler friction-drive rotary drum approach using salvaged and inexpensive components. The build requires no welding and uses a repurposed drill motor as the power source.
