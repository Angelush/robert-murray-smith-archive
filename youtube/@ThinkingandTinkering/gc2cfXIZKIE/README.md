## Overview

Robert demonstrates how to use Tinkercad, a free browser-based CAD tool, to modify existing 3D-printable STL files without needing to learn complex CAD software from scratch. Using a hand-crank generator part as a practical example, he shows how to cut, reshape, and add bolt holes to an existing design — turning it into a component suitable for a wind turbine, bicycle, or water wheel. The key takeaway is that 3D printing mastery comes from doing, not from exhaustive preparation.
