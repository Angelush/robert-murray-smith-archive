## Overview
Robert demonstrates how to use Tinkercad, a free browser-based CAD tool, to modify existing 3D-printable STL files without needing to learn complex CAD software from scratch. Using a hand-crank generator part as a practical example, he shows how to cut, reshape, and add bolt holes to an existing design — turning it into a component suitable for a wind turbine, bicycle, or water wheel. The key takeaway is that 3D printing mastery comes from doing, not from exhaustive preparation.

## Key Topics
- Robert's philosophy on learning by doing rather than front-loading knowledge
- Introduction to Tinkercad as a beginner-friendly, free, browser-based CAD tool
- Importing STL files into Tinkercad for modification
- Using shape tools as "cut" or "erase" tools (hole mode)
- Resizing and repositioning objects using the ruler/reference point tool
- Adding bolt holes to a modified cog/gear part
- Exporting the modified STL and loading it into Cura for slicing
- The iterative, progressive journey from printing others' designs to modifying and eventually creating your own

## Materials and Chemicals Mentioned
- PLA or similar filament (implied, not named specifically)
- STL file of a hand-crank generator component (crank handle/cog)

## Techniques and Methods
- Importing STL files into Tinkercad (file size limit ~23 MB)
- Using box/cylinder shapes set to "hole" mode as cutting/erasing tools
- Grouping objects to apply boolean subtraction (remove unwanted geometry)
- Export-and-reimport workflow to flatten grouped geometry into a single mesh
- Using the ruler tool with midpoint reference for precise placement and dimensioning
- Duplicating hole cylinders and adjusting coordinates to create a bolt-hole pattern
- Rotating parts to align geometry with reference axes before adding holes
- Exporting final STL and importing into Cura slicer

- **On learning style and pedagogy:** He acknowledges he personally champions kinaesthetic (hands-on) learning because he feels it's underrepresented, but stresses that learning is most effective when teaching style matches the learner's style — no single approach works for everyone.
