## Overview

Robert Murray-Smith demonstrates how to build a simple resistive heat pad using activated carbon felt and copper tape, inspired by a viewer suggestion for heated gloves. The pad can be powered by a standard battery and optionally fitted with a thermal breaker switch for safety. He runs multiple voltage tests showing the pad can reach 27–39°C depending on power input, making it practical for a wide range of warming applications.
