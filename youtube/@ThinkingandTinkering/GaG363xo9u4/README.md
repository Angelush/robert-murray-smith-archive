## Overview

Robert Murray-Smith demonstrates how to build a DIY solar kettle using an evacuated solar tube — a remarkably simple build that heats water to boiling in 20–30 minutes using sunlight alone. He notes that the same concept can be used as a portable solar shower by mixing the boiled water with cold water. He expresses puzzlement that a commercial UK version (which sold for £35) is no longer available.
