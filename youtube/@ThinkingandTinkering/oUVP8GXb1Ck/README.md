## Overview

Robert Murray-Smith introduces the Italian folk art tradition of Carta Pesta (papier-mâché sculpture) from Lecce in southern Italy, where it is used to create figurines and large public sculptures. In this first part, he demonstrates how to create a resin face cast from a doll's head using plasticine, resin, and flour as a filler, then refine it with carving and sanding.
