## Overview

Robert demonstrates a "floating rack" mechanism — a device that converts variable, bidirectional linear motion into consistent one-directional rotary motion — and shows how he designed and 3D-printed one using FreeCAD's Gears Workbench. The video highlights the floating rack as a practical tool for applications like wave and wind power generation, while also serving as a showcase for how accessible FreeCAD's gear design tools have become.
