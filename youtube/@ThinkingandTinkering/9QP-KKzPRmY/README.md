## Overview

Robert Murray-Smith demonstrates how to build a working wooden model of a piston-and-crank mechanism, sourcing inexpensive hardwood from kitchen chopping blocks rather than specialist suppliers. The video walks through cutting, assembling, and connecting all components — crankshaft discs, connecting rod, piston, and cylinder — to produce a functional model that converts rotary motion to linear motion. The main takeaway is that complex mechanical concepts can be illustrated beautifully and cheaply using basic woodworking techniques.
