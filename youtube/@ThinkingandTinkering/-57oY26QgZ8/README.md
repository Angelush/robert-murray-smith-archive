## Overview

This video is a competition entry submitted by a fan inspired by Robert Murray-Smith's channel, presenting the "Counterstar Wind Generator" — a vertical axis wind turbine built entirely from salvaged household junk. The core concept, drawn from Robert's own teaching, is that maximising the relative speed between stator and rotor increases generator output; the design uses a home-built planetary gear system to spin the stator counter to the rotor at 15× the blade speed. While the full wind-powered prototype did not function reliably due to gearing slip, the counter-rotating stator/rotor principle was validated and the device successfully lit an LED.
