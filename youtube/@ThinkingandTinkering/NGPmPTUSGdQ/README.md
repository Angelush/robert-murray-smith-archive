## Overview

Robert Murray-Smith builds a diesel punk lamp using a disassembled car gearbox donated by Canterbury Automotive, a candle lantern, and an LED tube that mimics the look of a nixie tube. Inspired by Ross from Intelligent Tinkering, the build is intentionally simple and customisable — the key steps are arranging the gears attractively, drilling a hole for a lamp holder, and welding on an M10 nut to secure it. The result is a striking piece of industrial art that Robert ultimately gifted back to Canterbury Automotive as a thank-you.
