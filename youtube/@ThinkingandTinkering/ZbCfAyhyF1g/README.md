## Overview

This video, presented by Luke on what appears to be Robert Murray-Smith's channel, demonstrates how to create a realistic wood-grain painting effect on surfaces. Using layered paint and wood stain varnish combined with texturing tools, the technique produces a convincing burr walnut appearance. The method was originally used on a steampunk-style PC case build (video 973) and is shown here in a dedicated tutorial.
