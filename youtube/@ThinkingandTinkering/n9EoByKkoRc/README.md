## Overview

This video demonstrates how to build a DIY light-up Christmas sign using entirely reclaimed and scavenged materials. Luke (presenting on the channel) walks through constructing a pine frame, mounting salvaged LED strips with a diffuser panel, and finishing with a stencil that conceals the sign's message until illuminated. The result is a decorative lit sign with a satisfying reveal effect.
