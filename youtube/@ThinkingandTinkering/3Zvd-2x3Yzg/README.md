## Overview

Robert Murray-Smith conducts a side-by-side speed comparison between the Elegoo Neptune 3 Pro and Neptune 4 Pro 3D printers, using a wind-powered zoetrope as the test print. The Neptune 4 Pro completed the print in roughly 1.5 hours versus 4 hours for the Neptune 3 Pro, with comparable print quality. The finished zoetrope — driven by wind via anemometer-style cups — successfully animates a running man strip.
