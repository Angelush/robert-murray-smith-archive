## Overview

Robert demonstrates an anti-twist wire mechanism as an elegant alternative to slip rings for connecting a home-built wind turbine's output wires to a battery storage system. Inspired by candle dancing folk traditions and an invention by Dale Adams (1971), the mechanism uses two counter-rotating gears to allow continuous rotation without the wires knotting. Robert highlights this as an underutilised solution that wind turbine makers appear to have overlooked.
