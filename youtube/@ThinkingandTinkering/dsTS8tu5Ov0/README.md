## Overview

Robert demonstrates a viewer-suggested idea (from a friend named Ian) for scavenging waste heat from central heating radiators using a large array of 10 Peltier devices. He tests the concept using an oil-filled electric heater as a stand-in for a gas-fired central heating radiator, then discovers a significant performance boost by adding Oasis floral foam soaked in water to the cold side, using evaporative cooling to maintain the temperature differential across the Peltiers.
