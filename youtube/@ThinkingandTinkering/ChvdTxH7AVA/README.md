## Overview

Robert Murray-Smith demonstrates how capillary action and water transpiration — the mechanisms trees use to pump water from roots to canopy — can be harnessed to create passive, pump-free evaporative cooling. Using floral foam (oasis) as a visual analogue for plant tissue, he shows how solar-driven evaporation produces a measurable temperature drop, and argues that this biomimetic principle can replace the pump in a conventional swamp cooler, significantly reducing energy costs.
