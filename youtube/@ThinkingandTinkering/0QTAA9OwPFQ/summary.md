## Overview
Robert Murray-Smith demonstrates a simple, math-based method for cutting wooden gears without specialist machinery. Using basic tools — a jigsaw, drill, compass, and protractor — he walks through the geometry of sizing a gear, spacing teeth, and finishing them with sandpaper. The key takeaway is that functional wooden gears are accessible to anyone with basic workshop tools and a willingness to do a little arithmetic.

## Key Topics
- Why he chose to make wooden gears instead of using the previous toothed-belt-on-disc method
- Deciding gear size and tooth size as a starting point
- Calculating circumference from tooth count and tooth spacing
- Using pi to derive gear diameter from circumference
- Marking tooth positions evenly using a protractor (every 30° for 12 teeth)
- Drilling holes and drawing tooth profiles using drill bits as guides
- Cutting out the gear with a jigsaw
- Sanding and finishing teeth with improvised sandpaper tools
- Testing gear mesh by screwing gears to a board and spinning them

## Materials and Chemicals Mentioned
- MDF (used as the gear blank material)
- 8 mm and 9 mm drill bits (used as sizing guides)
- 8 mm metal bar offcut (used as a tooth profile template)
- P60 grit sandpaper (for finishing teeth)
- Double-sided tape (to attach sandpaper to filing tools)
- Flat metal bar (improvised sandpaper file)

## Techniques and Methods
- Geometric gear layout: calculating circumference from tooth count × tooth pitch, then dividing by pi to get diameter
- Protractor marking: dividing 360° evenly by tooth count to place drill holes
- Drill-bit sizing trick: using an 8 mm tooth with a 9 mm hole to create 0.5 mm clearance on each side
- Jigsaw cutting of the gear blank
- Improvised sandpaper file (flat bar + P60 sandpaper) for flat tooth faces
- Dowel/bar sandpaper tool for rounding and evening tooth hollows
- Bench test: screwing gears to a board to check for binding before use
- Dremel for final finishing
