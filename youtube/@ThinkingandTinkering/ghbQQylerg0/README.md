## Overview

Robert Murray-Smith presents a clean geometric proof of the Pythagorean theorem (a² + b² = c²). He constructs a large square from four identical right-angled triangles arranged around a smaller central square, then equates two different expressions for the same total area to derive the theorem algebraically.
