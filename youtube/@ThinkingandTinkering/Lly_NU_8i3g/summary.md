## Overview
This video, presented by Luke (a contributor on Robert Murray-Smith's channel), demonstrates how to breathe new life into a slow, inexpensive old laptop by replacing Windows with Linux Mint (Cinnamon edition). The presenter walks through the entire process — from downloading the OS and flashing a USB drive to installing Linux Mint — showing that a £20 laptop can become a fast, capable workstation for students, kids, or general home use.

## Key Topics
- Buying and repurposing a cheap old laptop (£20)
- Linux as a free, lightweight Windows alternative
- Downloading Linux Mint (Cinnamon edition)
- Creating a bootable USB installer
- Navigating the BIOS boot menu to boot from USB
- Step-by-step Linux Mint installation (erasing the old Windows partition)
- Performance improvements after installation
- Suitability for students, kids, and budget home users
- Parental control options on Linux Mint (Mint Nanny, Firefox parental controls)

## Materials and Chemicals Mentioned
- Old laptop (purchased for £20, with a 250 GB hard drive)
- USB drive (used as the boot installer medium)

*(This is a software/computing video — no chemicals or lab materials are involved.)*

## Techniques and Methods
- Downloading the Linux Mint Cinnamon ISO from the official website
- Flashing the ISO to a USB drive using a dedicated flash tool (Balena Etcher implied by the "Flash" button described)
- Accessing the BIOS/boot menu (manufacturer-specific key; Escape in this case, F9 for boot device options)
- Booting into Linux Mint live environment from USB (GRUB menu)
- Saving files from the old hard drive before wiping
- Running the Linux Mint installer: selecting language, keyboard layout, Wi-Fi, multimedia codecs, and erasing the disk for a clean install
- Setting computer name and user password during installation
