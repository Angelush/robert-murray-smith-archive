## Overview

This video, presented by Luke on Robert Murray-Smith's channel, demonstrates how to create a large-scale tiled poster by splitting an image across multiple A4 pages using a free online tool, then printing, cutting, and assembling the sheets. The result is a full-scale printed image larger than a standard printer can produce, intended as a component for a future project on the channel.
