## Overview
This video, presented by Luke on Robert Murray-Smith's channel, demonstrates how to create a large-scale tiled poster by splitting an image across multiple A4 pages using a free online tool, then printing, cutting, and assembling the sheets. The result is a full-scale printed image larger than a standard printer can produce, intended as a component for a future project on the channel.

## Key Topics
- Using rasterbator.net to split an image into printable A4 tiles
- Configuring poster size and output settings
- Printing and downloading the tiled PDF
- Aligning pages using printed guide markings or a backlit LED panel
- Cutting and gluing the assembled sheets flat with a roller

## Materials and Chemicals Mentioned
- Glue (used to adhere the cut tiles together)

## Techniques and Methods
- Uploading an image via URL to rasterbator.net
- Configuring landscape orientation and custom output size (1.88 scale, ~52×37 cm)
- Selecting "no effects" style for a clean print
- Using a converted LED panel as an improvised light table for alignment
- Rolling over glued sections to ensure a flat bond
