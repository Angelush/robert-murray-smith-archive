## Overview

Robert Murray-Smith demonstrates how to produce methanol from deciduous wood using dry (destructive) distillation. By loading oak twigs into a simple paint-can retort, heating it, and passing the vapour through a copper coil condenser, he collects a water/methanol mixture. The key insight is that plants naturally contain methanol, which can be boiled off at 300–400°C and condensed into a usable liquid fuel.
