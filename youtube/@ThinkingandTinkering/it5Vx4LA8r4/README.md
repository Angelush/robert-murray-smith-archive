## Overview

A presenter named Luke builds a custom-sized wooden box lined with aluminium foil and aluminium tape to create a DIY Faraday cage capable of blocking all mobile phone signals. The key insight demonstrated is that the conductive lining must be electrically continuous — including bridging strips connecting the lid to the base — for the shielding to work. The completed box is tested live and shown to fully block incoming calls when closed.
