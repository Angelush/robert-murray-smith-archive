## Overview

This video demonstrates how to splice two lengths of jute rope together, a technique needed to create a rope of a specific length for a gravity generator project. The presenter (Luke) walks through a simple under-over weave method that produces a strong, neat braided join. The main takeaway is that with basic tools and the right technique, any two rope ends can be joined securely.
