## Overview

Robert Murray-Smith and his colleagues Steve build a DIY hotwire cutter (dubbed the "Megameltatron") using Kanthal resistive wire and salvaged parts from an old heater-stirrer. The video demonstrates the full assembly process and confirms the cutter works effectively on polystyrene, cutting clean circles for an ongoing project.
