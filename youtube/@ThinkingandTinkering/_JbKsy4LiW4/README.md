## Overview

Robert Murray-Smith demonstrates a simple gluing trick for bonding dissimilar materials — plastic to hardboard — using super glue accelerated with bicarbonate of soda. The combination creates a rapid, rock-hard bond and proves effective enough for structural use in a functional wind turbine model.
