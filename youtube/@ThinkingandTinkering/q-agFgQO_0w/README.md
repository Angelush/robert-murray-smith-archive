## Overview

Robert Murray-Smith demonstrates a 3D-printed bevel gear assembly that achieves gear reduction using gears of identical size — something conventionally considered impossible. Building on a previous 2:1 ratio design, he adds a second module to achieve a 3:1 ratio, and explains that the effect arises from contra-rotation as the bevel gear orbits around the assembly. The device doubles as a satisfying fidget toy and the files are shared on Thingiverse for remixing.
