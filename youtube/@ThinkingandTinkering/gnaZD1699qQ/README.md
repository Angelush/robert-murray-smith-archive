## Overview

Robert Murray-Smith demonstrates how to build a very low-friction magnetic bearing using simple, inexpensive materials. The bearing uses four repelling magnets to levitate a rotating axle assembly, with a ballpoint pen tip providing the single point of contact to constrain lateral movement. He concludes by noting the design is practical for sensitive motor research and similar low-friction applications.
