## Overview

Robert demonstrates using an inverted bicycle as a low-cost, versatile test bed for evaluating wind turbine generator configurations. Rather than building a full turbine and mounting it aloft only to discover problems, this setup allows rapid iteration on gear ratios and electronic controls. He shows live voltage and current measurements using a car alternator driven by hand-crank, validating that output voltage exceeds the excitation voltage fed into the rotor.
