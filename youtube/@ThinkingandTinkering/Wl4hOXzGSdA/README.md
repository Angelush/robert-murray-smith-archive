## Overview

Robert Murray-Smith demonstrates the NIF Differential Friction Drive, a mechanism patented in the 1930s that uses ball bearings as friction rollers to achieve high gear ratios with dramatically lower losses than traditional toothed gears. He 3D prints a working model using ping-pong balls as the rolling elements, explains the geometry behind the gear ratio, and contrasts this approach with the Perpetual Wedge (James White's design), highlighting where each excels and falls short.
