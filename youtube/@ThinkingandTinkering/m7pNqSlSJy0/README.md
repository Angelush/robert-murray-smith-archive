## Overview

This video demonstrates how to connect a resistive-paint heating panel (from a previous video) to an AC mains supply rather than DC. Robert walks through the components needed for a safe, temperature-controlled installation, then shows a simple test rig using a dimmer switch and a light bulb as a current limiter, achieving ~48°C at ~40W within seconds of powering up.
