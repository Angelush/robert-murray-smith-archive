## Overview

Robert demonstrates the safe procedure for discharging and disassembling a CO₂ fire extinguisher before scrapping it. The key safety point is that these extinguishers can retain dangerous pressure even when seemingly empty, so the valve must be safely released before unscrewing the head. The video is a brief practical guide aimed at scrappers and recyclers handling this type of equipment.
