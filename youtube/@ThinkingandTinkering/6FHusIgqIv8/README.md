## Overview

This video is a technical progress update on Robert's "SWAG generator" prototype — a wave-powered electrochemical energy device using graphite-coated plates as a gill-like electrode structure. Robert explains a key design refinement: isolating the electrode assembly from direct seawater contact using a closed tank and plunger system, which solves the marine fouling problem while allowing fluid experimentation. He also previews a companion video in which the same hardware is tested as a thermal battery using copper chloride solution.
