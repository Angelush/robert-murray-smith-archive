## Overview

Robert Murray-Smith demonstrates how to copper-plate organic keepsakes — specifically leaves and flowers — using conductive ink and a basic electroplating setup. The central lesson is that patience and low current are the keys to success: rushing the process burns off the conductive coating, while running at a few milliamps per square centimetre over 12+ hours produces a beautiful, detailed copper finish.
