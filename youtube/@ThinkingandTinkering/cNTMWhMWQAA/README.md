## Overview

Robert Murray-Smith demonstrates how to build a simple, low-cost hydrogen electrolyzer using stainless steel expanded metal mesh instead of the traditionally difficult-to-work stainless steel plates. The key innovation is electrolyzing a urea solution rather than water, which requires significantly less voltage (0.37V vs 1.23V), meaning more energy is recovered from burning the resulting hydrogen than was consumed in its production. The video is a practical build guide aimed at encouraging home experimentation with urea electrolysis.
