## Overview

Robert Murray-Smith designs and 3D-prints a spur gear differential (also called a "spur differential"), explaining how it differs from the more common bevel differential found in most cars. He models the entire assembly in Tinkercad using only basic primitives, then physically assembles the printed parts, demonstrating how the differential splits or combines rotational power between two output shafts while allowing them to turn at different speeds.
