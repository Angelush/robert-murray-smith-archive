## Overview

Robert Murray-Smith demonstrates how to make low-cost, paint-on resistive heating pads for his SILAR (Successive Ionic Layer Adsorption and Reaction) machine build, using Teflon-coated supermarket cooking sheets as the base. The key insight is that conductive ink can be applied directly to Teflon surfaces with a simple rubbing primer step, creating nearly invisible, effective heaters. He notes the technique is equally applicable to heated beds for 3D printers.
