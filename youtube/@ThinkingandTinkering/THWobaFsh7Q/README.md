## Overview

Robert Murray-Smith demonstrates how to build a rubber band-powered tractor toy using 3D-printed parts, building on his earlier rubber band motor video. He uses the project to explain the fundamental trade-off between torque and speed in geared systems. The video doubles as a practical lesson in prototyping philosophy — rubber bands are cheap, fast, and ideal for testing mechanical designs before committing to more expensive materials.
