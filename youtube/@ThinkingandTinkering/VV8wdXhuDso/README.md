## Overview

This is a one-week follow-up video testing conductive ink used as a replacement for thermal paste in a desktop PC. Luke runs CPU stress tests (Prime95) and a 3D graphics benchmark (3DMark) to evaluate temperatures, concluding that the conductive ink is performing well under sustained load with no thermal throttling, and plans a six-month long-term follow-up.
