## Overview
This is a one-week follow-up video testing conductive ink used as a replacement for thermal paste in a desktop PC. Luke runs CPU stress tests (Prime95) and a 3D graphics benchmark (3DMark) to evaluate temperatures, concluding that the conductive ink is performing well under sustained load with no thermal throttling, and plans a six-month long-term follow-up.

## Key Topics
- One-week real-world performance results of conductive ink as thermal paste
- CPU stress testing using Prime95 (torture test mode, ~15+ minutes)
- GPU/system benchmark using 3DMark
- Temperature monitoring with Speccy
- Comparison context: stock Intel cooler used throughout

## Materials and Chemicals Mentioned
- **Conductive ink** — used as a substitute for conventional thermal paste on a CPU
- **Thermal paste** — the conventional material being compared against (not reinstalled due to reluctance to disturb the existing setup)

## Techniques and Methods
- CPU stress/torture testing with Prime95 to measure peak and sustained temperatures
- System benchmarking with 3DMark for GPU-load temperature comparison
- Real-time temperature monitoring via Speccy
- CPU usage monitoring via Windows Performance Monitor
