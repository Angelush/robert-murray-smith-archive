## Overview

Robert Murray-Smith unboxes and tests the Elegoo Saturn 4 Ultra 16K resin printer, comparing output quality and fit tolerances directly against an FDM (filament) printer using the same model. He demonstrates that resin printing — particularly at 16K resolution — produces dramatically smoother surfaces and tighter mechanical fits (a working piston-and-cylinder assembly), while acknowledging that filament printing still has advantages in strength and workflow simplicity.
