## Overview

Robert Murray-Smith demonstrates Tesla's thermal magnetic generator (invented ~1929), a reciprocating engine that exploits the Curie effect — nickel loses its magnetism when heated, then regains it upon cooling, causing oscillating or rotary motion. He builds both the original oscillating version and a novel rotary variant, then scales it up using nichrome wire wound around a pulley wheel, showing the principle works at larger scale with a blowtorch as the heat source.
