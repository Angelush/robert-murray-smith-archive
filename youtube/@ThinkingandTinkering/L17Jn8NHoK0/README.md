## Overview

Robert Murray-Smith demonstrates how to salvage the motor and rollers from a broken belt sander for use in a homemade Van De Graaff generator. He disassembles the sander step-by-step, explaining what went wrong with it (an overheated, melted fan belt) and identifying the key components — rollers, motor housing, pulleys, and roller bearings — that can be repurposed. He notes that a new belt sander can be purchased for around £10 if you don't have an old one to strip.
