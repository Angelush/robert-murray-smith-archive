## Overview

Robert Murray-Smith demonstrates how to build a contra-rotating (counter-rotating) gear mechanism using simple peg gears cut from cutting boards. The video explains that turning a central gear causes two flanking gears to spin simultaneously in opposite directions, a principle with real-world applications in helicopters and drivetrains. The build is designed to be quick, robust, and accessible using basic workshop materials.
