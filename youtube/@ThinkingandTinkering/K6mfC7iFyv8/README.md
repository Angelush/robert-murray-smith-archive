## Overview

In this video, Robert tests a Mark II venturi turbine by combining the cone/pipe arrangement developed in video 1382 with an impeller salvaged from an exercise bike (previously used in his "ros turbine" from video 1126). Testing is done indoors using a fan due to no wind that day. The result demonstrates the concept works, generating measurable voltage from a microwave oven coil, and Robert outlines plans for a tighter-tolerance larger version.
