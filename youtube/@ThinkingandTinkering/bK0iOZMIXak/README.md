## Overview

This is a viewer competition entry submitted to Robert Murray-Smith and Luke, in which a maker from Nanaimo, British Columbia demonstrates two simple hand-cranked LED generators built from salvaged electronics found in the trash. The core insight is that the small DC motors inside DVD and VCR tray mechanisms can act as generators when turned manually, requiring almost no modification to produce usable electricity.
