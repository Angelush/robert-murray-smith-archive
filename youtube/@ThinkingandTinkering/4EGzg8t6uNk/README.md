## Overview

This video demonstrates how to create parametric cycloid shapes in Tinkercad's Code Blocks environment. Robert explains the mathematical equations behind epicycloids and shows how implementing the parametric equations directly in Tinkercad produces cleaner, more flexible results than his earlier geometric rotation method. The resulting cycloid disc can be exported as an STL file for 3D printing and used in epicycloid gear drives.
