## Overview
This video demonstrates how to create parametric cycloid shapes in Tinkercad's Code Blocks environment. Robert explains the mathematical equations behind epicycloids and shows how implementing the parametric equations directly in Tinkercad produces cleaner, more flexible results than his earlier geometric rotation method. The resulting cycloid disc can be exported as an STL file for 3D printing and used in epicycloid gear drives.

## Key Topics
- Epicycloid vs. cycloid geometry (circle around a circle vs. straight-line path)
- Parametric equations for generating cycloid curves
- Tinkercad Code Blocks as a parametric design tool
- Epicycloid gear drives and their applications in robotics
- Gear reduction and torque increase via compact epicycloid arrangements
- Exporting STL files from Tinkercad for 3D printing

## Materials and Chemicals Mentioned
None mentioned.

## Techniques and Methods
- Parametric design using epicycloid equations in Tinkercad Code Blocks
- Using count loops to step through degree increments and position circles along the cycloid path
- Grouping/cutting geometry in Tinkercad to produce a cycloid disc with pegs
- Exporting designs as STL files
- Earlier "backwoodsy" method: rotating wheels geometrically to cut the shape (referenced, not demonstrated here)
