## Overview
Robert Murray-Smith demonstrates a potentiometer painted with conductive ink, showing how it functions as a variable resistor. By touching a probe at different points along the painted strip, the resistance changes dramatically (from 1.8 kΩ down to ~140 Ω), which is used to control both LED brightness and motor speed via an Arduino. He frames this as a stepping stone toward building a fully painted circuit.

## Key Topics
- Conductive ink used as a painted potentiometer/variable resistor
- Resistance variation along a painted strip (1.8 kΩ to ~140 Ω)
- Controlling LED brightness using the painted potentiometer
- Controlling motor speed using the painted potentiometer
- Vision for a fully painted circuit as a future project

## Materials and Chemicals Mentioned
- Conductive ink (available from workingink.co.uk)

## Techniques and Methods
- Painting a resistive track with conductive ink to create a variable resistor
- Probing the painted strip at different points with patch cables to vary resistance
- Breadboard wiring for LED and motor control circuits
