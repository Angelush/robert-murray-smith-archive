## Overview

Robert Murray-Smith introduces spring motors as a practical, modular DIY component, demonstrating how a cheap steel recoil spring from a petrol chainsaw can be housed in a 3D-printed case to create a reusable spring motor module. He argues that spring motors remain relevant today — citing Trevor Baylis's famous wind-up radio as inspiration — and encourages makers to think creatively about applying this technology to solve problems. The finished module is available on Thingiverse.
