## Overview

Robert Murray-Smith demonstrates a modified planetary gear system where the ring gear has external teeth, allowing all three elements (sun, planets, ring) to be driven simultaneously — creating an infinitely variable transmission (CVT). He then explores applying this concept to wind turbines, showing how a self-regulating feedback mechanism using an idler gear can keep generator output relatively constant across a wide range of wind speeds.
