## Overview
Robert Murray-Smith demonstrates a modified planetary gear system where the ring gear has external teeth, allowing all three elements (sun, planets, ring) to be driven simultaneously — creating an infinitely variable transmission (CVT). He then explores applying this concept to wind turbines, showing how a self-regulating feedback mechanism using an idler gear can keep generator output relatively constant across a wide range of wind speeds.

## Key Topics
- Planetary gear system fundamentals (sun, planet, ring gears, planet carrier)
- Gear ratio calculations for planetary systems
- Modified ring gear with external teeth enabling dual-input drive
- Continuously variable transmission (CVT) using electronic motor control
- Application of the CVT concept to wind turbine speed regulation
- Self-regulating mechanical feedback loop using an idler gear
- Comparison to governor mechanisms; note that the system already exists in the Toyota Prius

## Materials and Chemicals Mentioned
- 3D-printed plastic gear components (designed in Tinkercad)
- Skate bearings (22 mm × 7 mm × 8 mm)
- 8 mm steel bar (axle)
- Washers
- Glue (for idler gear assembly)

## Techniques and Methods
- 3D printing of planetary gear components via Tinkercad and Thingiverse files
- Assembly of compound planetary gearbox with independently driven ring gear (external teeth added)
- Electronic CVT control by varying motor speed via variable resistor
- Mechanical self-regulation via idler gear feedback: faster input spins the ring gear in the opposing direction, counteracting output speed increase
- Physical demonstration using a flag as a visual output speed indicator

## Robert's Own Replies
- **Speed vs. torque trade-off:** Increasing output speed reduces torque; a low-torque (or appropriately matched) generator would be needed for any speed-increase variant.
- **Low-speed generators:** Notes that a low-speed generator implies high torque — always a balance between the two.
- **No interest in patents:** Explicitly states there is no point in patenting the idea for him; if companies are interested, they are welcome to it.
- **Prius confirmation:** Was pleased to learn from commenters that a similar planetary CVT system is already used in the Toyota Prius — takes it as validation he "isn't thinking crazy thoughts."
- **Governor analogy:** Clarifies that a traditional governor works by restricting fuel input, making it distinct from this mechanical feedback approach.
- **Perpetual motion misconception:** Directly corrects a commenter — a constant-velocity output does not lead to perpetual motion.
- **Practical advice:** Encourages people to build a physical model to understand gear ratio optimisation; "a model helps me to understand."
- **Generator heat:** Notes that running the generator too hard causes it to overheat, reinforcing why output regulation matters.
- **Disagreement welcome:** Emphasises openness to disagreement and that he is simply sharing what he built and what he thinks.
