## Overview
Robert introduces the Makera Carvera Air, a desktop CNC machine he received as a sponsored unit, framing the video explicitly for complete beginners with no prior CNC experience — exactly his own situation. Rather than a conventional review with pros/cons and a live demo, he sets up a multi-part series documenting his genuine learning journey from unboxing through CAD/CAM software to actually cutting material. His central takeaway is that CNC (subtractive manufacturing) meaningfully expands a maker's capabilities beyond 3D printing (additive), and that success requires stubbornness and willingness to learn, not just buying the machine.

## Key Topics
- What the video is and is not: an introduction for beginners, not an expert review
- Unboxing and physical overview of the Carvera Air and its accessories
- Context: Robert's background in 3D printing and desire to work with metal, wood, and plastic
- Comparison of additive (3D printing) vs. subtractive (CNC) manufacturing
- The full learning path required: CAD → CAM → machine operation
- Software choices: FreeCAD (chosen for its integrated CAM), alternatives like Fusion 360 and OpenShot mentioned
- Practical drawbacks: noise, dust/chips, bed size (300×200 mm), slow cutting speeds
- Workflow preference: USB file transfer rather than live PC connection
- Plan for a multi-part series covering the entire process

## Materials and Chemicals Mentioned
- **Aluminium** — machine body (solid cast aluminium frame); also a target cutting material demonstrated conceptually
- **Wood** — mentioned as a material the machine can cut, producing sawdust
- **Plastic** — cuttable material; plastic chips cited as a byproduct
- **8 mm rod** — mentioned as an example workpiece for the fourth-axis attachment

## Techniques and Methods
- **CNC milling / subtractive manufacturing** — removing material from a block using a spinning tool bit
- **Multiple light passes** — required due to the 200 W spindle; many shallow passes at high speed rather than deep cuts
- **CAD design** — using FreeCAD to create 3D models
- **CAM toolpath generation** — specifying tool size, path strategy, and material parameters within FreeCAD before generating G-code
- **G-code generation** — using Makera's standalone slicing software to convert CAM output to machine instructions
- **USB file transfer workflow** — loading G-code via USB stick rather than maintaining a live PC-to-machine connection
- **Laser engraving** — noted as an optional mode via the included 5 W laser module
- **Fourth-axis (rotary) machining** — mentioned as a future capability using the included mini-lathe attachment
