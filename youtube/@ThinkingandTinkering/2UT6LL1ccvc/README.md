## Overview

This video is a competition entry by "Jared from Creators Bench" for a challenge issued by Robert Murray-Smith to build a power generator from whatever materials you have lying around. Jared demonstrates a working generator built entirely from recycled/salvaged materials, and in an addendum reveals a second, more capable generator made from a modified ceiling fan. The main takeaway is that a functional electromagnetic generator can be built from scavenged parts with no precise component selection required.
