## Overview

Robert Murray-Smith disassembles a hand blender motor to explain how it works as a generator, then demonstrates a flywheel generator using 46 magnets arranged in a north-south configuration around a spinning case. He measures the output of a single coreless coil (~1.2W) and concludes that the real secret to generator performance is rotational speed, not exotic magnetic or coil arrangements.
