## Overview
Robert Murray-Smith unboxes and sets up the Qidi X-Max 3, a mid-to-high-range FDM 3D printer weighing ~38kg and costing approximately £1,000. He walks through the out-of-box assembly steps, initial calibration sequence, and key hardware features, concluding that the machine offers strong value for money given its engineering-grade material capabilities and build quality.

## Key Topics
- Overview of the current 3D printer market and price tiers (low/mid/high range)
- Unboxing the Qidi X-Max 3 and its included accessories
- Initial setup: removing shipping ties and bolts, language selection
- Head offset calibration using the included leveling pad
- Auto bed leveling (9×9 grid, 81 points)
- Input shaper calibration
- Filament loading procedure
- Touchscreen interface and Klipper firmware overview
- Wi-Fi connectivity vs. standalone USB operation
- Software: Pusher slicer (bundled on USB stick)
- Machine specs: print head speed (600 mm/s), acceleration (2,000 mm/s²), heated chamber, activated carbon filter

## Materials and Chemicals Mentioned
- **PLA Plus** — used for initial test filament load; half a kilo of black PLA+ included free
- **ABS** — discussed as now more viable due to the enclosed heated chamber and carbon filter
- **Carbon fiber-filled filament** — mentioned as a target engineering material; requires hardened steel hot end
- **Glass fiber-filled filament** — mentioned alongside carbon fiber as a use case for the hardened steel hot end
- **Metal-filled filament** — briefly mentioned as another engineering-grade option
- **Glue stick** — included in the box for bed adhesion if needed

## Techniques and Methods
- **Head offset / Z-offset calibration** — manual paper/plastic drag method to set nozzle-to-bed gap
- **Auto bed leveling** — 81-point (9×9) mesh mapping of the print surface
- **Input shaper calibration** — automated resonance compensation routine run at first startup
- **Filament loading via Bowden tube** — feeding filament through external tube, through runout sensor, to extruder
- **Hot end swapping** — replacing the stock copper hot end with the included hardened steel hot end for abrasive filaments
- **Standalone USB operation** — transferring print files via USB stick to avoid network resource usage
