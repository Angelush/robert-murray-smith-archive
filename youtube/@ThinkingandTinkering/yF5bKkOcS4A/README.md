## Overview

This short competition-entry video demonstrates a simple experiment combining two phenomena: an inductive flyback effect (a voltage spike produced when an inductor is disconnected) and a basic homemade motor built from a battery, magnets, and a bearing. The setup uses strips of tape to intermittently break the circuit, triggering the flyback spike to drive the motor. Robert closes with a caution that the battery is being run in a near-short-circuit condition, which is not recommended.
