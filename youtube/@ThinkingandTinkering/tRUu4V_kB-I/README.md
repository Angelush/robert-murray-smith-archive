## Overview

Robert Murray-Smith explores a UK startup called BPacks that has developed a bark-based alternative to conventional plastic fillers, using waste tree bark from the paper industry. The material is biodegradable, antibiotic, and compatible with existing thermoforming and injection moulding machinery, making industrial adoption significantly more viable. The video argues this is a meaningful step toward reducing single-use plastic waste.
