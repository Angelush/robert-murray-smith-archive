## Overview

Robert Murray-Smith builds a dry cell HHO (hydrogen/oxygen) electrolysis generator using graphite foil electrodes instead of the conventional stainless steel, inspired by a research paper showing graphite outperforms all other electrode materials tested. He constructs the cell almost entirely with scissors and a carpet knife, demonstrating it working with a 4% KOH electrolyte solution. The main takeaway is that graphite is simultaneously the best-performing, cheapest, and easiest electrode material for this application.
