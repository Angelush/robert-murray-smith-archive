## Overview

Robert demonstrates using a commercially available low-temperature Stirling engine (a "gamma type" thermal difference engine, purchased for ~£30 on Amazon) to test whether an evaporative cooler — made from water-soaked floral foam and black cloth — can generate a temperature differential sufficient to run the engine. The experiment succeeds with a 5°C difference between the hot (sun-heated) and cold (evaporating) sides, proving that solar evaporative cooling can drive a Stirling engine and thus generate electricity. Robert positions this as a practical, low-tool-barrier alternative to Peltier devices for small-scale off-grid power.
