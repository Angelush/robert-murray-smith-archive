## Overview

Robert demonstrates how to build a DIY RFID-blocking wallet using Grafoil as the core shielding material, protecting credit cards from contactless skimming. The build is simple and primarily cosmetic once the Grafoil layer is in place — the functional blocking is achieved with just the Grafoil and tape sandwich. He tests the finished wallet to confirm cards cannot be read by contactless systems.
