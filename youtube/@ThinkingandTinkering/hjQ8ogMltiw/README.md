## Overview

Robert Murray-Smith demonstrates how to make artificial muscles from ordinary nylon fishing line while on holiday at a seaside cottage, showing that meaningful science experiments can be done with minimal equipment. By twisting the fishing line under tension and heat-setting it with a hairdryer, the resulting coiled structure contracts when heated, lifting a 14-ounce weight nearly a full inch. The video's central message is that curiosity and imagination are the only truly essential tools for doing science.
