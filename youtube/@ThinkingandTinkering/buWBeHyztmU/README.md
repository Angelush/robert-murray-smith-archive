## Overview

Robert Murray-Smith demonstrates the Magnus effect using two simple DIY toys — a rubber-band-launched cardboard tube and a cut plastic bottle — showing how a spinning object generates lift. He explains the physics intuitively via Newton's third law and traces the phenomenon's real-world applications from Anton Flettner's 1920s rotor ship to modern wind turbines and ship stabilizers.
