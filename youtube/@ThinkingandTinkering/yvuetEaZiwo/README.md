## Overview

Robert Murray-Smith demonstrates how to build a Fluidyne — a liquid piston Stirling engine — using only a few pieces of plastic tubing, angle fittings, and copper pipe. The engine is heated with a blowtorch (or optionally a Fresnel lens for solar power) and requires almost no machined parts. The key takeaway is that a functional heat engine can be built cheaply and simply, and can even be converted into a linear generator with a cork, a magnet, and a coil.
