## Overview

Luke demonstrates how to make a DIY heavyweight body filler using only two ingredients: glass resin and zeolite. The mixture is used to fill gaps in a project piece, then sanded smooth after curing. The result is a clean, paintable surface that performs comparably to commercial body filler.
