## Overview

Robert Murray-Smith demonstrates how to build a flux switching alternator using parts salvaged from a microwave oven transformer. He disassembles the transformer, fabricates a six-pointed star rotor and two stator armatures from the laminated steel cores, winds coils from the transformer's primary wire, and adds permanent magnets to create an alternating magnetic circuit. The device successfully generates 0.4V, proving the concept works.
