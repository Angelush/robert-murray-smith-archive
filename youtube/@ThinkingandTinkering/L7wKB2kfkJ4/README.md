## Overview

Robert Murray-Smith replicates a "Flynn Device" (a motor based on designs by someone named Flynn) using salvaged transformer cores instead of the fan motor internals typically used. He walks through the disassembly and preparation of a 1:1 isolation transformer, splitting it into two halves and fitting neodymium magnets to create a functional magnetic circuit, ready for electrical testing.
