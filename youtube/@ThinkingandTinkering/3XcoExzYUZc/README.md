## Overview

Robert Murray-Smith demonstrates a "green" chemistry method for synthesizing copper nanoparticles using household chemicals and a blender. He reduces copper sulfate with ascorbic acid to precipitate elemental copper, then explores using a blender (exploiting cavitation, inspired by his graphene work) combined with an iron displacement reaction to produce a colloidal copper nanoparticle suspension. The video serves as an introductory proof-of-concept, highlighting that the resulting copper ink/fluid can be used for conductive traces, with photonic sintering as a recovery method if oxidation occurs.
