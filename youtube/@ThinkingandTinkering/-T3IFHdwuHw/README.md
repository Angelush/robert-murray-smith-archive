## Overview

Robert Murray-Smith demonstrates how to recycle ABS plastic from old computer cases and monitors into usable flat sheets. He walks through smashing, chipping, and pressing the plastic using basic tools and a heated steel plate. The result is a roughly 1mm thick sheet of reclaimed ABS suitable for making models or enclosures.
