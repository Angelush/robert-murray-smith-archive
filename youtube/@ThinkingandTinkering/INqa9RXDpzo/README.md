## Overview

Robert Murray-Smith demonstrates a full synthesis route for magnetite (Fe₃O₄) starting from common hardware-store chemicals, inspired by a viewer suggestion about magnetite's electrical conductivity. After testing resistivity and finding magnetite a poor conductor, he pivots to explore far more promising applications: using magnetite as the active coating in a DIY microwave furnace, sintering it into custom-shaped ferrite rods, and suspending it in oil to create ferrofluid.
