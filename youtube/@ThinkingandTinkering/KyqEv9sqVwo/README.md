## Overview

In this video, Luke (presenting on the channel) builds a hand-crank generator using an alternator salvaged from a hand-crank blender, following up on a previous exercise-machine generator project (video #1520). After struggling with belt slippage, he switches to a chain-and-sprocket drive, and ultimately measures approximately 3.35 watts output against ~1.5 watts input excitation, yielding around 1.8 watts net generation.
