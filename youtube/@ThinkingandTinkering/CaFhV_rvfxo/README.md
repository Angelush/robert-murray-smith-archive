## Overview

This short video, presented by Luke on Robert Murray-Smith's channel, demonstrates how to build a simple hand roller from a salvaged printer/laminating machine roller and basic hardware. The build is quick and low-cost, and the resulting tool is intended for rolling paper sticks but is versatile enough for screen printing, ink printing, laminating, and many other applications.
