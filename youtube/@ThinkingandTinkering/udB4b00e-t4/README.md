## Overview

Robert Murray-Smith demonstrates how to build a simple DIY air engine from scratch, explaining each major component — piston/cylinder, slide valve, scotch yolk crankshaft, and flywheel. He walks through both the construction and the underlying mechanical principles, emphasizing that the same concepts apply to converting a conventional engine to run on compressed air. The engine is ultimately powered using a vacuum cleaner as an air source.
