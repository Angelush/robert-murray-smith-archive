## Overview

Robert Murray-Smith uses the history of seat belt mechanisms as a springboard to explain how one-way (clutch) bearings work, demonstrating the principle with a physical model. He then walks through designing and 3D printing the key star-shaped inner component in TinkerCAD, assembling a functional one-way bearing using inexpensive off-the-shelf stainless steel shelf pins as the rolling elements.
