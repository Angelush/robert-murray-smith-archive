## Overview

Robert Murray-Smith visits a coastal Site of Special Scientific Interest on the London clay belt to introduce a DIY energy device based on zeolite clay. When dry zeolite (X13/zeolite 13X) absorbs water, it releases significant heat through an exothermic adsorption reaction; Robert harnesses this temperature differential using a Peltier thermoelectric device to generate electricity and run a small motor. The system is presented as safe, rechargeable by simply drying the clay, and built entirely from cheap, abundant, non-caustic materials.
