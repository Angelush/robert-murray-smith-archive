## Overview

A tribute video narrated by Robert Murray-Smith's son, documenting his father's attempt to build a water-powered over-balance wheel for a competition. The project explores why perpetual motion over-balance wheels fail (they reach equilibrium), then pivots to using water as the unbalancing element, ultimately powering an LED via a repurposed shake torch mechanism.
