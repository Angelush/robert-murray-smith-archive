## Overview

This video is a follow-up to Robert's earlier video on constructing a giant resistive radiator using conductive ink. Responding to viewer feedback, Robert moderates expectations for beginners by explaining that the process involves many uncontrollable variables — humidity, surface absorbency, application speed — that make first attempts very likely to fail. His core message is to start small, practice on smaller builds, and calibrate the ink's conductivity on your specific surface before attempting the full-scale version.
