## Overview

Robert Murray-Smith takes an ordinary aging desktop PC and rebuilds it inside a custom steampunk-styled enclosure. Motivated by a friend's steampunk computer build and his own previous steampunk speaker project, he walks through the complete process of stripping a standard tower PC, fabricating a new case from PVC foam board, and reassembling the components with decorative elements. The result is a fully functional steampunk computer with matching speakers, intended as an accessible DIY project for anyone with a basic home PC.
