## Overview

Robert Murray-Smith demonstrates how to build a touch-sensitive lamp entirely from upcycled materials — an old wine bottle and a wooden venetian blind — combined with a small commercial kit containing conductive ink, a touch switch, and copper tape. The finished lamp (nicknamed the "giraffe lamp") offers four touch-activated brightness levels and requires no specialist tools beyond basic woodworking equipment.
