## Overview

Robert Murray-Smith uses leftover bamboo poles from a previous project to build a bicycle frame, demonstrating that bamboo bikes are far simpler and cheaper to make than their commercial price tags suggest. He walks through cutting the bamboo poles to length using an existing bike as a template, fitting salvaged metal parts (head tube, axle bearings), and joining the frame using a peg-and-stitch lashing technique. The video ends with the bare frame assembled, with a follow-up planned to fit the remaining bike components.
