## Overview
Robert demonstrates a survival hack for cyclists: using reflective tape to spell out SOS in Morse code on a bicycle tire, so that passing car headlights will flash the distress signal. He acknowledges the idea is impractical — the tape wasn't very sticky or reflective — but presents it as a creative emergency concept. The video is short and self-deprecating in tone.

## Key Topics
- Emergency signalling using a bicycle as an SOS machine
- Morse code representation of SOS (three dots, three dashes, three dots)
- Reflective tape application to a bike tire
- Personal anecdotes about past cycling accidents and injuries (broken ribs, dislocated AC joint)

## Materials and Chemicals Mentioned
- Reflective sticky tape (described as the most reflective available in the shop, but noted as insufficiently reflective)

## Techniques and Methods
- Cleaning the tire surface before application to improve adhesion
- Applying short strips of tape for dots and longer strips for dashes in Morse code
- Using strip length to modulate reflection duration (longer strip = longer flash signal)
