## Overview

Robert demonstrates a survival hack for cyclists: using reflective tape to spell out SOS in Morse code on a bicycle tire, so that passing car headlights will flash the distress signal. He acknowledges the idea is impractical — the tape wasn't very sticky or reflective — but presents it as a creative emergency concept. The video is short and self-deprecating in tone.
