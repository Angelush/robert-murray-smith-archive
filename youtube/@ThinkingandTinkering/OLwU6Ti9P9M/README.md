## Overview

Luke (presenting on Robert Murray-Smith's channel) finds a Coomba 2242-7C cassette/CD recorder/player in a bin and attempts to repair it. He diagnoses and fixes several faults including non-working inputs, a cassette mechanism, and bad speaker solder joints, partially restoring the unit. The CD laser turns out to be dead beyond repair, but the cassette functionality — the main goal — is successfully restored.
