## Overview

Robert Murray-Smith demonstrates how to cut large circular disks from acrylic sheets salvaged from old LCD screens, using a homemade circle cutter built from printer parts. The video is a short, focused technique segment intended to complement a larger Wimshurst machine build series. He shows that a simple custom tool outperforms a chop saw for cutting acrylic without chipping.
