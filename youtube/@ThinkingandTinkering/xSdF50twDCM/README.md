## Overview

Robert demonstrates a fast, efficient method for removing tannins from acorns using a 40% ethanol-water solution, based on a published research paper. The process takes only three hours compared to traditional methods that can take days or weeks, opening the door to using acorns — an abundant, globally-produced resource — as both food and a non-food-crop source of bioethanol.
