## Overview

Robert Murray-Smith explores building a wind generator from a discarded household fan (a "taran"), inspired by the concept of the "wind wall" — an array of small turbines originally conceived by New York designer Joe Duet. He argues that accessibility and the ability to build from recycled materials matters more than peak efficiency, and demonstrates a working prototype using salvaged fan components paired with a cheap stepper motor.
