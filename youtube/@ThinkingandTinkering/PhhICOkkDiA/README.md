## Overview

Robert Murray-Smith explains the geometry behind cycloidal gear drives and demonstrates how to design a custom cycloid disc from scratch using Tinkercad, including its built-in Code Blocks scripting environment. The key insight is that gear ratios are fundamentally about distance traveled along a surface — not tooth count — and that a cycloid profile can be generated by iteratively "chiseling" small arcs from a disc using an eccentric cam shape. The video culminates in a printable cycloidal disc produced entirely within Tinkercad's free browser-based tools.
