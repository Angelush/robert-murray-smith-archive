## Overview

This video is an update to Robert's earlier methanol room heater (video 1702), addressing the main complaint viewers had: difficulty sourcing the carbon felt wick used in the original design. He demonstrates a simpler replacement wick built from an old sock, stainless steel mesh, and standard food tins, concluding that it works just as well and is far easier to make from common household items.
