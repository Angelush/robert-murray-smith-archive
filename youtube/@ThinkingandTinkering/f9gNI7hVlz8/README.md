## Overview

Robert Murray-Smith demonstrates how to build a simple, low-cost vegetable oil burner/heater using cheap or repurposed materials. The core innovation is a perforated metal strapping used as a wick holder, which draws in air alongside oil to produce a surprisingly smokeless flame. The video is aimed at people seeking affordable, DIY heating solutions, particularly using waste cooking oil.
