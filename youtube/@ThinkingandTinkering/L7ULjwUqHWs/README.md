## Overview

A short woodworking tutorial demonstrating how to reinforce mitre/butt joint frames using wooden dowel. The presenter (Luke) shows a simple two-step technique: drilling 45-degree angled holes through each corner joint and gluing dowel rods into them, resulting in a dramatically stronger frame. A clever folded-card trick is highlighted as the key tip for reliably gauging the 45-degree drill angle.
