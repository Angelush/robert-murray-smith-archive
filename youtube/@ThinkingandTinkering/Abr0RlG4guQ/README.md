## Overview

In this short video, Robert Murray-Smith demonstrates an improved method for electrically connecting carbon fiber to copper conductors for use in nanocarbon fiber heating elements. Building on a previous video where solder was used (limited to ~150°C), he shows how crimping provides a more durable, high-temperature connection good up to ~400°C — the point at which carbon itself begins to burn.
