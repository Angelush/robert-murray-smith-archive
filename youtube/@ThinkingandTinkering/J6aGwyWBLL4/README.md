## Overview

Robert Murray-Smith demonstrates a simple, homemade memristor built using graphene oxide as the active layer sandwiched between two aluminium electrodes. By reversing the polarity of a 12V battery connected to the device, he shows a measurable and stable resistance change of ~30 ohms, illustrating the core memristive property of resistance memory. He presents this as a proof-of-concept "paintable" memristor ink system worth further exploration.
