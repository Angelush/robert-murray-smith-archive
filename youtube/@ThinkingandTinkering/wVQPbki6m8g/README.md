## Overview

Robert Murray-Smith demonstrates how to make graphitic carbon nitride (g-C₃N₄) using a home kiln, starting from cheap urea pellets. The video highlights that this remarkable photocatalytic material — capable of splitting water into hydrogen using sunlight — can be synthesized at home for pennies per gram despite costing ~$100/gram commercially. The process is simple enough to serve as a first experiment for anyone with a kiln.
