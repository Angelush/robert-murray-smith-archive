## Overview

Robert Murray-Smith demonstrates how to convert a salvaged induction motor (pulled from an upright fan) into a switched reluctance generator, making it suitable for use in a wind turbine without requiring a gearbox. He modifies the motor's iron stator and rotor by cutting away poles, adding permanent magnets, and rewinding new coils to create a device that generates electricity as the rotor alternately opens and closes the magnetic flux path. The key takeaway is that switched reluctance generators are extremely cheap, durable (wear only on bearings), and well-suited for low-speed wind turbine applications.
