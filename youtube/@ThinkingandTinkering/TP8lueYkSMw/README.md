## Overview

Robert Murray-Smith demonstrates how to build a super-sensitive relay from scratch, using a hand-wound coil, a hacksaw blade as the switching arm, and neodymium magnets to create a balanced magnetic field. Unlike a standard relay, this design allows the trip voltage to be precisely tuned by adjusting the position of the magnets, and can even be extended to act as a multi-switch that triggers different contacts at different applied voltages.
