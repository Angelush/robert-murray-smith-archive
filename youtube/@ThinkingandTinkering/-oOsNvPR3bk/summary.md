## Overview
Robert demonstrates a surprisingly powerful feature in Tinkercad — the "hole" tool — which allows any 3D object to be instantly converted into a mold by inverting it and enclosing it in a solid box. Using a Bauhaus chess set as the example, he shows how grouping a "holed" object with a bounding box produces a ready-to-print or manufacturable mold. His main takeaway is that this technique democratises small-scale casting by eliminating the need for expensive commercial molds.

## Key Topics
- The Tinkercad "hole" tool and how it inverts any shape into a void
- Grouping objects to merge a hole-set model with a bounding box to produce a mold
- Importing any STL file into Tinkercad and converting it to a mold
- The economics of commercial injection molds vs. 3D-printed molds for small production runs
- Creative casting applications: chocolate chess pieces, Easter eggs, resin decorations, small-batch products

## Materials and Chemicals Mentioned
- **Chocolate (dark and white)** — suggested as a fun casting material for chess pieces
- **Resin** — mentioned for casting and selling small-batch decorative items
- **Ceramic filament** — noted in comments as a suitable filament for printing molds that need heat resistance
- **Ethyl acetate / acetone** — briefly raised in comments in the context of smoothing PLA vs. ABS prints (Robert clarifies ethyl acetate is for PLA, acetone for ABS)

## Techniques and Methods
- Setting grouped 3D objects to "hole" mode in Tinkercad to invert them into negative space
- Enclosing the holed object in a bounding box, then grouping both to generate a one-piece mold
- Importing third-party STL files (e.g., from Thingiverse) into Tinkercad for the same workflow
- 3D printing the resulting mold directly
- Sending the mold file to an external manufacturer (CNC machining or other processes)
- Consideration of a two-part mold design for more complex shapes (mentioned in comments as planned future work)
