## Overview

Robert Murray-Smith demonstrates how to build a belt drive system for his paper bicycle project. Rather than using a conventional chain and pedals, he constructs a large rear drive cog by repurposing a bicycle wheel rim and a timing belt, mimicking the drivetrain design of an e-scooter. The result is a low-cost, fabricated belt-drive sprocket bolted directly to the rear wheel.
