## Overview

Robert demonstrates a joule thief circuit built around a Tesla bifilar pancake coil instead of the conventional ferrite toroid ring. He wires it up in a slight variation of the standard dual-thief configuration — placing the 1K resistor between the joined coil ends rather than at the transistor base — and successfully lights an LED from a single 1.5V battery. He notes that because the coil is known to resonate, this substitution is particularly interesting.
