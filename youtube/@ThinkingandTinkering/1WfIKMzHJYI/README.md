## Overview

In Part 5 of his budget 1kW wind turbine series, Robert Murray-Smith spends a day experimenting with different coil configurations using microwave oven transformer (MOT) coils, testing how core presence, orientation, wire gauge, and magnet coverage affect power output. His main takeaway is that there is no single "correct" answer — coil design is a set of trade-offs between voltage, amperage, wind speed range, and efficiency.
