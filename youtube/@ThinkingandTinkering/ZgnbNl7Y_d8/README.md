## Overview

Robert Murray-Smith demonstrates how to build a DIY spin coating machine from a repurposed magnetic stirrer and a chocolate tin, then uses it to apply graphene oxide (GO) to glass substrates. The video shows the full process from construction through coating, resulting in a thin, transparent, electrically conductive graphene layer — confirmed by measuring approximately 200 ohms resistance across the coated glass.
