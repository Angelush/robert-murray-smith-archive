## Overview
Luke demonstrates how to create a custom 3D-printable lighter case by merging multiple STL files in Chitubox without any CAD software. The project is a Legend of Zelda-themed Clipper lighter sleeve made as a Christmas gift, combining a base case with decorative shield, sword, and rupee elements. The finished print is painted with black spray paint and white acrylic to replicate the in-game color scheme.

## Key Topics
- Design philosophy: considering style, utility, and the recipient before starting
- Sourcing and selecting free STL files for a themed project
- Merging/combining multiple STL files in Chitubox slicer software
- Resizing, rotating, and positioning multiple objects relative to a base piece
- Slicing and printing the combined file
- Post-print finishing: IPA wash, black spray paint base coat, white acrylic detail painting

## Materials and Chemicals Mentioned
- IPA (isopropyl alcohol) — used to clean the resin print after printing
- Black spray paint — used as a base coat over the entire print
- White acrylic paint — used to pick out raised decorative details before adding color

## Techniques and Methods
- Importing multiple STL files into Chitubox
- Scaling and rotating STL files to fit a base object
- Using the "move > centered" function to align objects on an axis
- Manually adjusting X/Y/Z axis positioning to group files together as one printable assembly
- Slicing the combined layout for resin 3D printing
- Base coat + detail painting workflow (black undercoat, white acrylic highlight, then color fill)
