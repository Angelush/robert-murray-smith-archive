## Overview

Robert Murray-Smith demonstrates a 3D-printed clamp built around a hypocycloid gear mechanism, following up on a previous video about hypocycloid gear ratios. The key insight is that hypocycloid gears are nearly impossible to back-drive, making them ideal for a clamp that requires little effort to close but holds with tremendous force. He also suggests the same design could serve as a non-slip drawing compass.
