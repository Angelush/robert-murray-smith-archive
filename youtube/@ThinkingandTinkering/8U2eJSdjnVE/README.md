## Overview

Robert Murray-Smith explores the chemistry behind Victor Petrik's method of making graphene through explosive exfoliation. He demonstrates a series of increasingly energetic reactions using potassium permanganate as an oxidant with various fuels, showing how reaction speed can be controlled to function as a chemical fuse — ultimately igniting a pre-intercalated graphite mixture to exfoliate it into graphene.
