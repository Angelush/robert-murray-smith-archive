## Overview

Robert demonstrates a proof-of-concept magnetically levitated, near-frictionless wind turbine bearing inspired by the ballpoint pen bearing from a 1994 video. Using 3D-printed parts, a 12mm steel ball bearing, and neodymium magnets, he constructs a low-friction pivot that keeps the rotating element stable without wobbling. The build culminates in a small wind turbine that successfully lights an LED when blown upon, validating the concept.
