## Overview

Robert Murray-Smith explores hidden and experimental features within the Cura 3D printing slicer, focusing on the "wire printing" mode — a technique that prints only the wireframe skeleton of a model, inspired by early 3D pens. He explains that wire printing was removed from Cura 5.2 but remains accessible in earlier versions such as the Elegoo Cura build (based on Cura 4.8), and offers practical advice on how to get better results from it.
