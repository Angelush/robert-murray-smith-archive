## Overview

This short tutorial demonstrates how to heat-bend plastic into precise right angles using a simple wooden jig and a heat gun. The presenter (Luke) explains the technique as a superior alternative to bending aluminium, which proved difficult to work with cleanly. The result is a crisp, accurate bend that fits the intended project component far better than the metal attempt.
