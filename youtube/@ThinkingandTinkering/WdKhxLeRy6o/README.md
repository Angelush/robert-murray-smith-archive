## Overview

Robert Murray-Smith scans two medieval wooden grotesques (often called gargoyles) mounted on historic buildings in Canterbury, UK, which he explains were originally used as signs for brothels serving pilgrims. He processes the 3D scans using free and accessible software tools to clean, align, and extract the faces, then 3D prints them. The key takeaway is that historic street art can be digitally preserved and reproduced using consumer-grade scanning and printing tools.
