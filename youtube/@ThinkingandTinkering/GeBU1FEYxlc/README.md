## Overview

Robert Murray-Smith outlines his plan to generate electricity from a rocket stove by building a low-tech, accessible Stirling engine. He evaluates Peltier devices and steam engines as alternatives before settling on a diaphragm-style alpha Stirling engine, which avoids the need for a lathe or precision machining. The core goal is to create something cheap, simple, and buildable by anyone with minimal tools.
