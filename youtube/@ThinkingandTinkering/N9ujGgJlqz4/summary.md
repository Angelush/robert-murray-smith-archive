## Overview
Robert demonstrates how to build bespoke custom machinery using T-slot aluminium extrusion profiles, requiring only minimal tools. As a practical example, he salvages the punch mechanism from a broken comb binder and builds a complete multi-hole punch machine around it in approximately two hours, making the case that aluminium extrusion is an accessible, affordable framework system for anyone wanting to fabricate custom machines.

## Key Topics
- T-slot aluminium extrusion as a low-skill, minimal-tool framework system for custom machine building
- Upcycling/salvaging a broken comb binder's punch mechanism as the functional core of a new machine
- Assembly techniques using cast angle brackets and T-slot nuts
- Building a multi-hole hole punch machine from scratch
- Potential future upgrades: motorising the punch with solenoids, DC motors, feed rollers, and Arduino control

## Materials and Chemicals Mentioned
- **T-slot aluminium extrusion (20×20mm profile)** — structural framework material, ~£5.15/metre (UK supplier: aluminium-profile.co.uk)
- **Cast angle brackets** — used to join extrusion sections at corners
- **T-slot bolts and nuts** — slide into the extrusion channel to secure brackets
- **Acrylic sheet** — used for the feed table surfaces and handle

## Techniques and Methods
- Assembly of T-slot aluminium extrusion frames using angle brackets and sliding T-nuts (no drilling or sawing required)
- Repurposing/upcycling an existing mechanism (comb binder punch unit) as the functional core of a new machine
- Sizing and fitting extrusion lengths by hand (screwdriver, Allen key, pliers only)
- Adding acrylic panels as table surfaces bolted to the extrusion frame
