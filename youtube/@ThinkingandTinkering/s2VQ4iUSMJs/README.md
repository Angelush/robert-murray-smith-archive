## Overview

This video demonstrates how to make a simple smoke flare (smoke bomb) using a three-ingredient pyrotechnic mixture packed into a short PVC pipe casing with a homemade fuse. The presenter walks through mixing, packing, and igniting the device, achieving a large smoke output. The main takeaway is that packing more active material into the casing produces more smoke.
