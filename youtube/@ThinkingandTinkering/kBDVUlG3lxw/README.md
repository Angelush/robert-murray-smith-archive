## Overview

This video demonstrates how to generate electricity directly from chalk by combining quicklime (burnt chalk) with water to produce heat, then harvesting that heat using a Peltier device salvaged from a mini fridge. It is a follow-up to video 1267, which used the same reaction to run a Stirling engine, and shows that the thermoelectric approach yields a measurable electrical output (~75 mA at ~2 V) without moving combustion parts.
