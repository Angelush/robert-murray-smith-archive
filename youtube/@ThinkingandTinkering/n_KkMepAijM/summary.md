## Overview
Robert Murray-Smith demonstrates how to design professional gear sets in Tinkercad using only three built-in shape primitives. He covers spur, helical, herringbone, and bevel gear types, then walks through building a complete rotary-to-linear motion mechanism — a cam-and-rack assembly that converts a spinning input gear into back-and-forth rack movement — which he makes available on Thingiverse for 3D printing.

## Key Topics
- Introduction to Tinkercad's built-in gear primitives (metric gear, "Crema" rack, "PR gear")
- Creating a spur gear with matching rack (rack-and-pinion)
- Creating helical gears and helical racks using the inclination angle parameter
- Creating herringbone gears and racks by mirroring and merging helical gears
- Brief overview of helical bevel gears
- Step-by-step construction of a 7-part rotary-to-linear motion mechanism
- Cam design with yoke/slot for converting rotation to linear travel
- Using indexing holes and workplanes for accurate part alignment
- Practical 3D printing workflow with files shared on Thingiverse

## Materials and Chemicals Mentioned
None mentioned.

## Techniques and Methods
- Using Tinkercad's Shape Search to find gear primitives
- Setting gear parameters: module size, tooth count, bevel angle, helical angle
- Aligning objects using Tinkercad's Align tool
- Mirroring shapes to create herringbone gear halves
- Using the Workplane + D-key drop technique to align parts flush to a face
- Grouping and ungrouping objects to combine solids and holes
- Creating a cam-and-yoke slot using rounded-end cutouts (semicircle primitives)
- Using hexagonal holes/shafts for drive axle connections with tolerance fitting
- Creating indexing holes in a base plate for accurate multi-part assembly registration
- Adjusting snap grid resolution (1 mm → 0.1 mm) for fine positioning
