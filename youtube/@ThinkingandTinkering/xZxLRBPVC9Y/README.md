## Overview

This short tutorial demonstrates a simple cardboard transfer technique for accurately marking hole positions before drilling, solving the common problem of misaligned holes when attaching hardware. The presenter (Luke) shows how pressing a hardware component through cardboard with a hammer creates a precise template that can then be used to transfer exact hole locations to any workpiece. The method ensures a perfect fit every time with minimal tools.
