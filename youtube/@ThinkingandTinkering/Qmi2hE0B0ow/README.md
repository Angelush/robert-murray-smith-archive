## Overview

This is a competition entry submitted to the Robert Murray-Smith channel, in which a presenter named Dan demonstrates a prototype he calls a "two tube generator." The device uses donut-shaped tubes containing spinning neodymium magnets to generate electricity, with bridge rectifiers used to condition the output. It is an early-stage prototype, and Dan acknowledges it needs further development.
