## Overview
Robert Murray-Smith demonstrates how to repurpose a broken overhead projector (OHP) into a homemade episcope (opaque projector) using an upturned plastic bin as the housing. He replaces the original halogen lamp with an LED floodlight module, wires in additional LED floods for illumination, and mounts the original optics on top — producing a functional tracing/opaque projector for under a few pounds. The video concludes with a live projection test, showing the device works well enough to project and trace images onto a wall.

## Key Topics
- History and terminology of projection devices: diascope, episcope, epidiascope
- Attribution to mathematician Euler (used one in 1756)
- Decline of overhead projectors in favour of document cameras
- Repurposing an OHP into an episcope/tracing projector
- Replacing halogen lamps with modern LED alternatives
- Basic UK domestic electrical wiring for lighting circuits
- Practical use cases: tracing projectors, murals, image projection

## Materials and Chemicals Mentioned
- Halogen lamp (original OHP lamp, ~£2–3, noted as dying out)
- LED floodlight unit (600 lumen, used as primary light source after disassembly)
- LED work clamp light (700 lumens)
- LED torch (1300 lumens)
- Thermal paste (used on LED unit reflector)
- Plastic bin (40p, ~30×30×30 cm, used as projector housing)
- 1.5 mm twin-core-and-earth cable (for lighting circuit wiring)
- Plastic conduit (for cable management)
- Terminal blocks (for connecting wires)

## Techniques and Methods
- Disassembly and cleaning of a broken OHP
- Extracting the LED module from a commercial LED floodlight
- Retrofitting the LED board into the original OHP lamp housing using existing fittings
- Cutting holes in a plastic bin using a multi-tool and hole saw
- Basic UK ring/lighting circuit wiring: collecting neutrals, switching the live, earthing
- Single-switch wiring to control all lights simultaneously
