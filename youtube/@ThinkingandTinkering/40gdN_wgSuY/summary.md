## Overview
In this short update video, Luke demonstrates a painted circuit featuring four LEDs controlled by an Arduino, firing in sequence with a 300-millisecond delay to create a rhythmic chasing light effect. The circuit uses conductive ink traces branching from a single ground line to drive the LEDs from digital pins 10–13. This is presented as a stepping stone toward a larger goal of building mood-lighting picture frames with dimmable, sequenced LEDs.

## Key Topics
- Painted circuit construction with an Arduino
- Sequential LED animation using timed delays in code
- Circuit layout using jumper leads connected to Arduino digital pins
- Future project direction: mood-lighting picture frames with potentiometer brightness control

## Materials and Chemicals Mentioned
- Conductive ink (used to paint the circuit traces)
- LEDs (four units)
- Jumper leads (five total)
- Arduino (microcontroller)

## Techniques and Methods
- Painting circuits with conductive ink to create branching trace layouts
- Sequential LED control via Arduino sketch using timed delays (300 ms between each LED)
- Connecting a single ground rail that branches into four LED paths
- Wiring LEDs to Arduino digital pins 10, 11, 12, and 13
