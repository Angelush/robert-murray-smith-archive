## Overview

Robert tests whether homemade biodiesel (made in a previous video, #1672) can be used as a fuel in a standard primer-type kerosene/paraffin stove. After a lengthy startup process of 3–4 minutes instead of the stated 30 seconds, the biodiesel successfully vaporizes and burns, demonstrating that with patience it can work in existing kerosene equipment. The main takeaway is that biodiesel's higher viscosity requires significantly more heat to vaporize, but once it does, the self-sustaining combustion works well.
