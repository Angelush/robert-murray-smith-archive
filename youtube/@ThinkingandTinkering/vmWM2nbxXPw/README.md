## Overview

Robert builds a large-scale vibration cleaner using a floor sander (a "Tomcat" orbital sander) as the base, welding a steel cradle frame to hold interchangeable Tupperware containers. The device is designed to clean heavy metal parts using his "wonder solution" combined with gravel as an abrasive medium, scaling up a technique he had previously demonstrated with a small hand sander.
