## Overview

In this video, Robert Murray-Smith adapts a tea towel-based vertical axis wind turbine (built in video 1353) to more closely match an original folding wind turbine design submitted by a viewer named Ron Ratting. The adaptation involves adding stays and extra arms to create a three-bladed rotor with fabric sails, resulting in a collapsible, portable wind turbine. Robert tests it outdoors and begins planning a further evolution toward an H-Darrieus configuration with NACA airfoil blades.
