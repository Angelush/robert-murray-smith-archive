## Overview

This video demonstrates how to fabricate a custom motor mount for an e-scooter using repurposed aluminium shelving brackets. The presenter cuts, bends, and bolts the bracket material into a clamp that hugs the motor securely, then adds feet for mounting to the scooter frame. The result is a lightweight, low-cost motor mount finished with a copper coat and draft sealant to dampen vibrations.
