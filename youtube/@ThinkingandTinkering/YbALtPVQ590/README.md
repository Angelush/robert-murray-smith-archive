## Overview

In this video, Robert Murray-Smith motorizes a quad bike he built in previous episodes (106.3 and 106.6) using bicycle parts and old shelving. He adapts a vintage DC 12V scooter motor to drive the quad via a chain-and-sprocket transmission machined from a bicycle bottom bracket assembly. The episode ends with a successful test of the drive system, with a speed controller still to be fitted before a full test run.
