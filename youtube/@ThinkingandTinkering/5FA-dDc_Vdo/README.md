## Overview

Robert demonstrates how to build a pseudo-magnetic levitation bearing for a flywheel using salvaged speaker magnets. He explains the different types of magnetic levitation — static, dynamic, and diamagnetic — then focuses on pseudo-levitation as the most practical DIY approach, showing how it can support surprisingly heavy loads (e.g., a 2.5 kg weight) with very low friction, making it ideal for long-spinning energy storage flywheels.
