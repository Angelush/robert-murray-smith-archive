## Overview

This short tutorial video covers the final wiring steps for the motor assembly of a homemade Van de Graaff generator. Robert demonstrates how to correctly connect the power supply wires to a dimmer switch (identifying the common and L1/live-out terminals), close up the housing, and perform a basic test run showing variable speed control. The key takeaway is that while the wiring is straightforward household electrics, Robert cautions that basic electrical confidence is essential given the voltages involved.
