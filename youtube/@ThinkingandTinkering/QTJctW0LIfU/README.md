## Overview

Robert Murray-Smith builds a functional electric bicycle frame almost entirely from rolled newspaper tubes, demonstrating that paper can serve as a viable structural material for a bike frame. He assembles the frame using butt joints wrapped with carbon fiber and resin, then adds an unusual belt-drive system using a timing belt glued to an oversized rim. The finished paper e-bike successfully carries him around a car park under its own power using a 120W motor.
