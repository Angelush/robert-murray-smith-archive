## Overview

In this video, Mandy (a colleague at Robert Murray-Smith's lab) demonstrates how to make a Harry Potter-style wand using a chopstick, string, and a hot glue gun, then copper-plate the handle using electroplating. The key enabling material is a water-resistant conductive ink developed by the team, which makes the non-conductive wand surface ready for electrodeposition. The finished wand receives a shiny copper-plated handle after approximately seven hours in the electroplating bath.
