## Overview

Robert Murray-Smith documents his very first print using the Elegoo Mars 2 Pro UV MSLA resin 3D printer, walking through the complete process from setup to finished part. He explains the key mechanical principles at work — particularly the "tug of war" between the build plate and FEP sheet — and demonstrates the preparation steps that led to a successful first print on the first attempt. The printed object is later revealed to be a part for a mini wind turbine.
