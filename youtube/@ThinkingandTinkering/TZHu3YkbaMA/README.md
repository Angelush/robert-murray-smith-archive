## Overview

Robert builds a rubber band heat engine mounted on the magnetic floating bearing system introduced in the previous video (1428). The engine exploits rubber's unusual thermomechanical property — contracting rather than expanding when heated — to generate rotation. The result works in principle but runs at an extremely slow speed (~2 rotations per day with only four rubber bands).
