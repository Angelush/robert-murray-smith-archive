## Overview

This video is a promotional preview for an upcoming live-stream battery-making seminar hosted by Robert Murray-Smith. Robert reassures viewers that no chemistry expertise is needed — the process is designed to be done in a kitchen with minimal equipment. The kit sold through his shop contains all the necessary chemicals and components to make a Prussian blue analog battery material with an energy density comparable to lithium-ion batteries.
