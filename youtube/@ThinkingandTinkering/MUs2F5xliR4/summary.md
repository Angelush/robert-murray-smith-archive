## Overview
A short walkthrough video showing how to install the Cura slicing software included with the Elegoo Neptune Max 3D printer. Robert emphasises that the process is far simpler than most people expect, encouraging viewers to just get started rather than overthinking it. The core takeaway is that from unboxing to first print can be achieved in under 30 minutes.

## Key Topics
- Robert's philosophy of "just give it a go" rather than overthinking new tasks
- Installing Elegoo's bundled Cura software on Mac and Windows
- Installing USB/serial drivers included on the SD card
- Adding the Elegoo Neptune Max as a printer profile in Cura
- Loading an STL file and slicing it for printing
- Thingiverse as a resource for finding STL files

## Materials and Chemicals Mentioned
None mentioned.

## Techniques and Methods
- Installing Cura on macOS: drag DMG contents to the Applications folder
- Installing Cura on Windows: run the EXE installer and follow prompts
- Adding a non-networked printer in Cura by selecting Elegoo > Neptune Max
- Drag-and-drop import of STL files onto the Cura print bed
- Basic slicing workflow: load STL → adjust scale/position → slice → print
