## Overview

This video is a fan competition entry for Robert Murray-Smith's "Made From Paper" challenge, showcasing a marble machine built almost entirely from paper and cardboard. The machine uses a mousetrap-powered governor and spool mechanism to drop a ball, which triggers a sign reading "Sub to TNT." The creator documents the design process and troubleshooting steps, including slowing the governor by adding a rubber band to reduce mousetrap power.
