## Overview

Robert Murray-Smith demonstrates a "green" (non-toxic) method for reducing graphene oxide back to graphene using vitamin C (ascorbic acid) instead of the conventional but hazardous hydrazine. The process was inspired by his earlier work reducing copper oxide to copper metal with ascorbic acid. The result is a stable black colloidal dispersion of graphene in water suitable for further experiments.
