## Overview

Robert Murray-Smith demonstrates how to build a simple electrostatic motor using his floating pen kit as the core component. The motor uses a high-voltage generator to create a corona discharge that spins a rotor made from aluminium foil strips on a paper sleeve. The build is intentionally minimal and accessible, requiring very few materials and tools.
