## Overview

This video demonstrates a variable reciprocating rack mechanism that converts back-and-forth linear motion into continuous one-way rotation, developed as part of Robert's ongoing variable hypocycloid project. The key feature is that the output rotation is proportional to the stroke length, making it fully variable. The video progresses through two iterations — a 3D-printed ratchet design to a cleaner version using industrial one-way clutch bearings — ultimately achieving quiet, accurate, zero-backlash motion conversion.
