## Overview

This short tutorial demonstrates how to desolder and remove components from a circuit board using a heat gun. The presenter (Luke) successfully removes toroid components from a board to reuse them in another project, emphasizing that iron-core components like toroids survive the process well, while silicon components are likely to be damaged by the heat required.
