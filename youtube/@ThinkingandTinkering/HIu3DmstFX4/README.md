## Overview

Robert Murray-Smith demonstrates how to build a simple water-filled magnifying glass lens inspired by Josh Silver's self-adjusting eyeglasses, which use water pressure to change focal length. Using uPVC builders board (Cetraboard in the US), double-sided acrylic gel tape, and a drinking straw as a fill port, he assembles a biconvex lens chamber in just a minute or two. The result is a surprisingly effective, glass-like magnifying lens that can be made cheaply and quickly at home.
