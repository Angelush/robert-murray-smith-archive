## Overview

Robert demonstrates how to create a homemade plastic by combining PVA glue (a mix of polyvinyl acetate and polyvinyl alcohol), borax solution, glycerol as a plasticizer, and an inert filler such as talcum powder. The borax cross-links the polymer chains, producing a moldable putty that hardens into a tough, white plastic as it dries. The video is an exploratory experiment inspired by a friend's question about magnetic silly putty, showing that different fillers yield dramatically different material properties.
