## Overview

Robert demonstrates how to convert a cheap pillar drill (drill press) into a slot cutter by fitting a router bit and bolting an angle iron guide to the table. He then uses the slotted timber to build a linear peg gear mechanism — a rack-and-pinion style device drawn directly from Pollum's Mechanical Alphabet — illustrating how basic tools can be adapted creatively when a full machine shop is out of reach.
