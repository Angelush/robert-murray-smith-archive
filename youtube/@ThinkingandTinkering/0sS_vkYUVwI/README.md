## Overview

In this short-form video, Robert Murray-Smith demonstrates the production of graphene via electrochemical exfoliation of intercalated graphite using a welder (arc/MMA welding). Starting with approximately 5 millilitres of intercalated graphite, the process achieves a remarkable 150× volume expansion into 750 millilitres of exfoliated material.
