## Overview

Robert Murray-Smith demonstrates how to build a simple, adaptable axial flux generator using 3D-printed discs, neodymium magnets, and a serpentine copper coil. The design uses only a few parts, is lightweight, tolerant of manufacturing error, and can be coupled to wind turbines, hand cranks, bike gearing, or water wheels. A demonstration shows it producing 60 volts with a planetary gear hand crank.
