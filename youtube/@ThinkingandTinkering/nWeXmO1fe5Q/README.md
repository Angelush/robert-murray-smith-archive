## Overview

This is a quick response video addressed to viewer Terry Hull, demonstrating how to cast thin, rigid bioplastic sheets using casein (a milk protein). Robert explains the full process from dissolving casein in an alkaline solution to hardening it with formaldehyde, and shows a ~25 micron transparent sheet as a proof of concept. The video highlights casein's versatility as a substrate for functional materials, including conductive films.
