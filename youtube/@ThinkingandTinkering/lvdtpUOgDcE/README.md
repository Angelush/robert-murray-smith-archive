## Overview

Robert demonstrates the history, construction, and practical applications of peg gears, showing how they can be built with minimal tools (hole saw, chop saw, sander, drill). The central build features two cage gears and one partial peg gear arranged to convert continuous rotary motion into reciprocating (back-and-forth) motion, which he argues has a wide range of useful applications.
