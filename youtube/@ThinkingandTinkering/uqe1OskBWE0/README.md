## Overview

Robert tests a cable drum wind turbine built from scrap wood, following a suggestion from viewer John Wynn. The turbine uses 62 ceramic magnets mounted around the rim of a cable drum, with a salvaged microwave oven transformer acting as the pickup coil. Despite being crudely constructed, it successfully generates enough power to light a 36-watt LED panel in approximately 10–12 mph wind, demonstrating that rim-mounted low-torque generation is a viable approach.
