## Overview

Robert Murray-Smith designs and 3D prints a blisk (blade + disk) turbine inspired by the V2 rocket engine, using TinkerCAD and a NASA turbine blade image as a reference. The video covers the historical context of the V2's engineering, a step-by-step CAD walkthrough, and a live spin test, with plans to add neodymium magnets and a coil to turn the device into a small generator.
