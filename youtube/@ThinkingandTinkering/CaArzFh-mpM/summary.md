## Overview
Six months after applying conductive ink as thermal paste in his main PC, Luke revisits the build to run a series of benchmark and real-world load tests. The results show that temperatures have remained stable and consistent with earlier tests, demonstrating that the conductive ink has neither improved nor degraded over time. The conclusion is that conductive ink is a viable long-term thermal interface material.

## Key Topics
- Six-month follow-up on conductive ink used as CPU thermal paste
- Prime95 stress test and temperature results
- 3DMark benchmark and CPU temperature monitoring
- Gaming load tests (CS:GO and Rocket League) as thermal benchmarks
- Video rendering as a real-world CPU load test
- Overall assessment of conductive ink's longevity and thermal performance

## Materials and Chemicals Mentioned
- **Conductive ink** — used as a thermal paste/thermal interface material between CPU and cooler

## Techniques and Methods
- **Prime95 stress test** — run for ~15 minutes to push CPU to maximum load and record peak temperatures
- **3DMark benchmark** — GPU/CPU benchmark with built-in temperature reporting
- **Gaming load testing** — running CPU-intensive esports titles (CS:GO, Rocket League) at max settings while monitoring temperatures
- **Video rendering test** — exporting a 5-minute 1080p/60fps video while logging CPU temperatures
- **Speccy** — system monitoring software used to track CPU temperatures during tests
