## Overview

Robert Murray-Smith demonstrates how to wind a bifilar pancake coil using a homemade acrylic jig, walking through the construction process step by step. He uses twisted-strand wire and explains the bifilar winding technique — where two wires are wound together and then connected so the current travels in one continuous loop. The video is a practical introduction before he plans to use the coils in a jewel thief circuit experiment.
