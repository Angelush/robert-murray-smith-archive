## Overview

Robert Murray-Smith builds a pair of steampunk-style speakers inspired by a friend's steampunk computer project. Starting with cheap £5 USB speakers, he fabricates gramophone-style flared cones from PVC board using a petal/trapezoid construction method, mounts them on bent copper towel-holder brackets, and produces a functional and visually striking result. The video demonstrates that creative DIY aesthetics can be achieved with minimal cost and simple materials.
