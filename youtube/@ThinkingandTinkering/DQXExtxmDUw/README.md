## Overview

A competition entry responding to a challenge (from Robert and Luke) to generate enough electricity to light an LED. The presenter built a pendulum-based energy harvester using permanent magnets, inductors salvaged from a photocopier, and a voltage multiplier circuit. The LED was ultimately lit — both via the pendulum system and a backup three-cell potato battery.
