## Overview

Robert Murray-Smith demonstrates how to make an organic, biodegradable gelatin foam, inspired by prosthetics maker Michael Davey who uses gelatin instead of latex for theater prosthetics. The foam is plastic, flexible, castable, and reheatable, making it a surprisingly versatile material. This is the first in a two-part series; the second video uses the foam to construct a "jelly capacitor."
