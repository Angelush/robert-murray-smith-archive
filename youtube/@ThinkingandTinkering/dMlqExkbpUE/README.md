## Overview

A short Christmas advent calendar video (Day 13) in which Robert demonstrates how to build a simple balloon hovercraft from a CD and a balloon. The balloon is glued to the CD so that air escaping from it creates a cushion, lifting the CD off a flat surface and allowing it to glide freely.
