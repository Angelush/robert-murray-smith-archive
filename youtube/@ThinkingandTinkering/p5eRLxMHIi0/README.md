## Overview

Robert Murray-Smith demonstrates a redesigned transparent, flexible solar cell built on a plastic substrate, using graphene as an internal charge-transport network alongside a photoactive material and nanofibrious cellulose binder. The device is fabricated by doctor-blading a blended mixture onto copper and aluminum electrode strips, exploiting their different work functions for current directionality. Testing in sunlight produces measurable photocurrent (~9 microamps), confirming the proof-of-concept despite modest output.
