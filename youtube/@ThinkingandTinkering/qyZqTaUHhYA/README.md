## Overview

Robert Murray-Smith builds a compact DIY hybrid air conditioner using reusable freezer blocks and a 3D-printed housing, powered by a 5V USB fan. The device pre-chills air by blowing it over frozen ice packs (rather than ice), producing cool, dry air without dripping or humidity issues. It can run from a USB port, rechargeable battery, or solar panel, making it portable and off-grid capable.
