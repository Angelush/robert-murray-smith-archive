## Overview
This video is the second in a two-part series reviewing the Aufero Laser 2, a diode laser engraver/cutter sent to Robert for review. Robert focuses on the software side of the machine, demonstrating how to set it up and get it running using Laser GRBL and LightBurn. His main takeaway is that Aufero's decision to support standard, community-backed open software rather than proprietary solutions makes the machine far more accessible and capable for a wide range of users.

## Key Topics
- Software options for the Aufero Laser 2: Laser GRBL (free) and LightBurn (commercial, 30-day trial)
- Compatibility with older and newer versions of Windows
- Installation process — straightforward "yes yes yes" installer approach
- The "home" lock quirk in both Laser GRBL and LightBurn and how to unlock the machine
- The value of open/community-supported software vs. proprietary laser software
- Analogy to Arduino's community-driven success
- Machine capabilities: speed (~10,000 mm/min max), cutting 4mm plywood, 3–5mm acrylic, engraving on glass

## Materials and Chemicals Mentioned
- 4mm plywood (cutting capability)
- 3–5mm acrylic (cutting capability)
- Glass (engraving capability)

## Techniques and Methods
- Laser engraving and cutting using diode laser
- Importing design files (JPEG, PDF, PNG, BMP) into Laser GRBL or LightBurn
- Unlocking the machine from the software home-lock state before operation
- Designing in external programs and importing into laser control software
