## Overview

Robert demonstrates wand (brush) electroplating as a simpler alternative to bath plating, using zinc as the plating metal on conductive HDPE plastic. The video is a preparatory step for a separate project building a "sunflower motor," showing how wand plating can metallize plastic surfaces with minimal equipment and cost.
