## Overview

In this video, Luke demonstrates how to repurpose a discarded washing machine into a functional vertical axis wind turbine (VAWT) using only the components found inside the machine itself. He disassembles the washing machine, cuts and bends fins from the stainless steel drum, and connects the salvaged motor to serve as a generator. The turbine successfully spins in the wind by the end of the video, with electrical output testing deferred to a follow-up video.
