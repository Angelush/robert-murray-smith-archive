## Overview

Robert Murray-Smith demonstrates a homopolar generator built from microwave parts, then investigates a counterintuitive result: when the magnets and conducting disc are rotated together — with no relative motion between them — the device still generates measurable current (~4 microamps). This challenges the conventional expectation that current requires relative motion between a conductor and a magnetic field, and turns out to be an example of the Faraday paradox.
