## Overview

Robert Murray-Smith introduces the transverse flux machine — a generator/motor topology combining radial and axial magnetic flux paths — which has been around since 1968 but remains largely unknown. He explains how it works using C-shaped silicon steel stator pieces and a toroidal coil, then builds a working prototype from salvaged motor laminations and cheap neodymium magnets, demonstrating it generating enough power to light an LED headlight. The key takeaway is that transverse flux machines are more compact, efficient, and gearless than conventional designs, making them highly attractive for wind turbines and electric vehicles.
