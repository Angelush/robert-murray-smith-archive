## Overview

This is the fourth entry in Robert Murray-Smith's "Impossible Gears" series, exploring how same-sized gears can still achieve a gear reduction. In this episode, he replaces the lever arm used in the original spur gear design with a rotating cage mechanism, demonstrating a clean 2:1 reduction using only spur gears. He assembles the 3D-printed mechanism on camera and verifies the ratio by marking dots on the input and output gears.
