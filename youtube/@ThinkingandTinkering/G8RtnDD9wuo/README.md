## Overview

Robert Murray-Smith demonstrates how to make a pyrotechnic fuse using an imitation Sugru mixture. He combines potassium permanganate, sulfur, and silicone sealant (with cornstarch added to accelerate curing) to create a flexible, slow-burning rubber cord. The fuse is then used to successfully ignite a thermite reaction, confirming its practical effectiveness.
