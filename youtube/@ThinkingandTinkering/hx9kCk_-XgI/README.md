## Overview

Robert Murray-Smith demonstrates how to adapt an outdoor rocket stove for indoor use by constructing a steel box enclosure that acts as a baffle chamber, forcing exhaust gases to recirculate before venting up a house chimney. The modification is presented as simple and inexpensive, using basic sheet steel and welding. The key takeaway is that any rocket stove can be converted for indoor use as long as exhaust gases are baffled before reaching the flue.
