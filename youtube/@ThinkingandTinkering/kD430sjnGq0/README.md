## Overview

This video demonstrates how FWG's water-resistant conductive ink can be used to make non-metallic surfaces electroplatable with copper, using minion erasers as the demonstration object. Presented by Mandy from FWG with a technical segment by Robert, the video walks through the full process from surface preparation to a finished, polished copper-coated minion. The key takeaway is that low current density applied slowly over a long period (two days at 50mA) produces the best results.
