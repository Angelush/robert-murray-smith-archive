## Overview

Robert Murray-Smith highlights the environmental problem of toothbrush waste — 200 million discarded annually in the UK alone — and explains why mixed-material construction makes them nearly impossible to recycle economically. He then demonstrates an inventive upcycling solution: using toothbrush heads as the bristle elements in a 3D-printed ratchet/freewheel mechanism, turning a landfill-bound object into a silent, surprisingly forceful mechanical component.
