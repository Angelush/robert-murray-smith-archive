## Overview

Robert Murray-Smith demonstrates a novel synthesis of copper nanoparticles using an insoluble copper salt (red copper oxide) reduced in a gelatin solution with formic acid. The gelatin acts as both a capping agent and dispersant, producing water-dispersible copper nanoparticles suitable for photonic sintering inks used in printed conductive electronics. Robert notes this synthesis approach was original to him and had been improperly used without credit in academic papers.
