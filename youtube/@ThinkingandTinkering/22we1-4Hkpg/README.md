## Overview

Robert demonstrates a novel technique for creating a "printed circuit board" using FWG graphene-based conductive ink coated onto PET film, which is then laser-cut to define the circuit traces. The target circuit is a stylophone, requiring only two active components (a piezo and a 555 timer IC) with all passive components embedded in the ink traces themselves. The experiment succeeds, producing a working stylophone from the laser-cut conductive film.
