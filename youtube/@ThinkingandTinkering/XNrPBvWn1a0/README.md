## Overview

Robert Murray-Smith begins a multi-part series building a rideable plastic trike entirely from uPVC builders board (roofline plastic), demonstrating the material's remarkable strength and versatility. He constructs the main chassis using H-shaped girders made from flat uPVC sheet, attaches a heavy-duty swivel castor wheel for the rear pivot, and rigs a rear-wheel steering mechanism using bending plates and push rods. The video serves as both a practical build log and an enthusiastic showcase of what this widely available, easy-to-work plastic can achieve.
