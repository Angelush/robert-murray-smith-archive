## Overview

Robert builds a low-cost DC generator by combining the gear train from a salvaged exercise bike with a printer motor, achieving approximately 12V at 2–3 amps output. The core insight is that universal motors (such as those from vacuum cleaners) require high rotational speed to generate useful power, so the multi-stage belt-and-pulley system from the exercise bike provides the necessary gear ratio. The finished unit can be converted into a wind turbine by attaching blades to the input shaft.
