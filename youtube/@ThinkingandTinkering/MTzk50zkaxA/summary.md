## Overview
Robert demonstrates how to solve a scaling problem in Tinkercad when using copy-and-rotate to place holes around a rotor design. He shows that the "Your Creations" tool, accessed via the basic shapes dropdown menu, allows you to save a custom shape with locked scaling so it maintains its exact dimensions as it is duplicated and rotated into position.

## Key Topics
- The scaling bug when duplicating and rotating shapes in Tinkercad
- Designing a generator rotor with evenly spaced magnet holes
- Using the "Your Creations" feature to save custom shapes
- The "Lock Part Size" / "Don't Scale" option when creating a shape
- Practical use cases for "Your Creations" beyond this specific fix (chain links, ball-and-socket joints, etc.)

## Materials and Chemicals Mentioned
- Neodymium (or similar) cylindrical magnets — 10 mm diameter, referenced as fitting into 10.2 mm holes to allow for print variance

## Techniques and Methods
- Creating a rotor base from a cylinder primitive (100 × 100 mm, 5 mm deep)
- Using a hole primitive sized at 10.2 mm to create a magnet pocket
- Merging shapes to cut holes in Tinkercad
- Duplicate-and-rotate workflow for repeating features around a circle (22.5° increments)
- Saving a custom shape via "Your Creations → Create Shape" with "Lock Part Size" enabled to prevent scaling on rotation
