## Overview

Robert Murray-Smith demonstrates a home-built implementation of a flux-switching transformer based on a 1968 patent (US3368141) by Sunto Gar, which uses a permanent magnet to assist switching action in a transformer core. The experiment uses a mechanically oscillating relay to drive the transformer and compares output voltage with and without neodymium magnets in place. The magnets nearly double the output — from ~300 mV to ~500 mV — confirming the patent's claim that the permanent magnet enhances flux switching.
