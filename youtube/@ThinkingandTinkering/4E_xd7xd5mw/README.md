## Overview

This video is a competition entry submitted by a fan/follower of Robert Murray-Smith, inspired by Robert's work on DIY green energy systems. The creator built a small wind turbine using salvaged vinyl LP records as rotor blades, a converted fan motor as a generator, a bridge rectifier, and a capacitor — successfully lighting an LED from wind-generated power. It serves as both a project showcase and a heartfelt tribute to Robert's influence as a science communicator.
