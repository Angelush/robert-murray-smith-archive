## Overview

This short tutorial demonstrates how to create a stained glass effect on clear acrylic using common household materials. The presenter (Luke) traces a printed template using a black acrylic/PVA glue mixture to simulate lead lines, then colors the sections with dry erase markers and Sharpies for a translucent look, sealed with clear lacquer. The finished panel is applied decoratively to the side of a computer case with convincing results.
