## Overview

Robert Murray-Smith demonstrates a DIY bicycle-powered dual-purpose machine that simultaneously pumps water and charges a USB device at 5V. Using a salvaged pump, a three-phase motor, a bridge rectifier, and a buck converter, he builds a simple system driven by a bicycle on an exercise stand with minimal parts.
