## Overview

Robert Murray-Smith demonstrates how to build low-cost, functional machine-head tuning pegs for a homemade banjo ukulele using salvaged and inexpensive hardware. The design is inspired by headless guitar tuning systems and uses wing nuts sliding in aluminium heat-sink channels to create a screw-tensioned string holder. The finished set of four tuners costs around £5 and took about two hours to make including filming.
