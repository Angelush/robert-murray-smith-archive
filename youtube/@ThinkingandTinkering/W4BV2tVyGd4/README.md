## Overview

Robert demonstrates how to set up Linux (specifically Linux Mint 21.2 Cinnamon) as a cost-effective alternative to expensive hardware for 3D printing and video work. He walks through installing the Cura slicer via the package manager and configuring it for the Elegoo Neptune 4 Pro. The main takeaway is that Linux can breathe new life into older, inexpensive machines — he runs his setup on a £100 Dell Precision T5600 server from eBay.
