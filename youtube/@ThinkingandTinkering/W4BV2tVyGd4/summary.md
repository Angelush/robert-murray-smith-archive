## Overview
Robert demonstrates how to set up Linux (specifically Linux Mint 21.2 Cinnamon) as a cost-effective alternative to expensive hardware for 3D printing and video work. He walks through installing the Cura slicer via the package manager and configuring it for the Elegoo Neptune 4 Pro. The main takeaway is that Linux can breathe new life into older, inexpensive machines — he runs his setup on a £100 Dell Precision T5600 server from eBay.

## Key Topics
- Why Robert chose Linux over a more expensive computer setup
- The role of slicer software in 3D printing (STL to G-code conversion)
- Installing Cura on Linux Mint via the built-in package manager
- Adding a custom printer profile for the Elegoo Neptune 4 Pro in Cura
- Configuring start/end G-code and extruder settings for the Neptune 4 Pro
- Running DaVinci Resolve on repurposed server hardware

## Materials and Chemicals Mentioned
- Generic PLA (used as the filament material for a test print)

## Techniques and Methods
- Installing software via Linux Mint's graphical package manager
- Creating a custom printer profile in Cura (non-network printer, custom G-code)
- Slicing an STL file and exporting to file for transfer to the printer
- Adjusting print speed (from 60 to 120 mm/s) for the Neptune 4 Pro
