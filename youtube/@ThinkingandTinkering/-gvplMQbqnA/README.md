## Overview

Robert Murray-Smith demonstrates how to build a diaphragm (rolling/top-hat) piston using 3D-printed parts and a finger cut from a latex rubber glove, bypassing the tedious sanding normally required to achieve a good piston fit. The key insight is that the rolled rubber sleeve unrolls like taking off a sock, creating a near-frictionless seal with very low hysteresis. The result is a working piston that responds to even low pressures and can be incorporated into pumps, compressors, or engines.
