## Overview

Robert Murray-Smith, collaborating with Ross from "Intelligent Tinkering," explores a method for adding printed designs to 3D prints using laser jet toner transfers. The technique exploits the fact that laser toner fuses at ~130°C while PLA prints at ~220°C, causing the toner to bond preferentially to the hot plastic. The result is a simple, accessible way to add decorative imagery to 3D-printed objects without changing filament or post-painting.
