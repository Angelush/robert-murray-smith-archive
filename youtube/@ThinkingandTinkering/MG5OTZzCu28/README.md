## Overview

In this video, Luke (presenting on Robert Murray-Smith's channel) attaches a belt drive system and electronics to a bamboo tadpole trike. The build uses a 500W motor with a 16-tooth drive cog paired to a 136-tooth rear wheel cog, achieving roughly an 8:1 gear ratio. The drive system works well in a car park test run, and Luke notes he plans to reuse the same drive setup in a future quad build.
