## Overview

Robert Murray-Smith demonstrates how to build a simple wood tongue drum in approximately two hours using recycled pine bed slats as keys and a repurposed speaker box as the resonator body. The video emphasizes accessibility and low cost, showing that a functional musical instrument can be made from salvaged materials with basic tools. The key design insight is using individually bolt-on keys, which simplifies both construction and tuning.
