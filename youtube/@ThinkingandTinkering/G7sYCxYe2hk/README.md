## Overview

Robert Murray-Smith demonstrates a practical technique for salvaging large electrolytic capacitors from old UPS (Uninterruptible Power Supply) circuit boards. The challenge is that high-current PCBs use thick copper planes that act as heat sinks, making conventional desoldering impossible. His solution is to use a saw to cut the PCB around each capacitor pin, reducing the copper thermal mass to a manageable size before desoldering.
