## Overview

Robert Murray-Smith demonstrates a simple, homemade magnetic switching element — a device that can turn a permanent magnet's external field on and off using electromagnetically reversed flux. By combining a neodymium magnet core with a wound copper coil and an H-bridge relay, the assembly can be switched from zero attraction to strong attraction, or vice versa. The device is presented as a building block for a larger axial flux magnetic-assisted motor project.
