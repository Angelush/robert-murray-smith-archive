## Overview

Robert Murray-Smith demonstrates a passive solar evaporative cooling effect using oasis foam and carbon felt — the same materials used in a previous water purification experiment. He observes a striking 17°C temperature drop (from 32°C ambient down to 15°C) at the wet foam surface and proposes this as the basis for a low-energy, solar-powered air conditioning system using a split-system design with circulating fluid pipes.
