## Overview

Robert Murray-Smith builds a DIY UV curing box from inexpensive fluorescent light fittings purchased at a local hardware store, replacing the standard tubes with UV fluorescent equivalents. The project is motivated by his work developing a conductive flexographic printing ink that requires UV curing. He demonstrates that a functional UV cure box can be built for around £20, compared to the hundreds of pounds a commercial unit would cost.
