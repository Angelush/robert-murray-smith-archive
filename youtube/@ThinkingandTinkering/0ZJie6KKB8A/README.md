## Overview

Robert demonstrates how to build a "safety heater" — an alcohol burner that automatically extinguishes if knocked over, by applying the principle of the Davy lamp. The design uses a stainless steel mesh as a flame arrestor to separate the fuel reservoir from the open flame, meaning the flame cannot travel back through the mesh to the fuel source. The result is described as one of the safest possible DIY burners.
