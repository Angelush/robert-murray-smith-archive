## Overview

Robert Murray-Smith builds a hank drum using two aluminium woks as an accessible alternative to the traditional propane tank method. He cuts vibrating tongues into one wok using angled hole saw cuts, joins both woks together using an aluminium welding rod, and demonstrates the resulting bell-like tone. The video concludes that a hank drum is achievable without welding equipment or a propane tank.
