## Overview

Robert Murray-Smith demonstrates how to intercalate graphite using a sulfuric acid/nitric acid solution, then exfoliate it into expanded graphite nanoplatelets as a precursor to graphene. The process involves soaking large-flake graphite overnight in the acid mixture, then rapidly expanding it via direct heat (torch) or microwave. The result is a dramatic volumetric expansion that can then be sonicated or shear-mixed to produce graphene or graphene nanoplatelets.
