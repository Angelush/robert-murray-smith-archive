## Overview

This is a short announcement video by Patty Murray-Smith (Rob's wife) and Emily from FWG, previewing an upcoming live event on 24 August 2019. The event will cover the differences between batteries, capacitors, supercapacitors, and fuel cells, with live demonstrations from Rob (plant fuel cell) and Steve (hydrogen fuel cell). The video also briefly showcases new electroplating solution kits and a forthcoming children's lab kit.
