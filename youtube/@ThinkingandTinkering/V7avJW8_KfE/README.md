## Overview

Robert Murray-Smith explores the often-overlooked role of transmission systems in power engineering, then builds and demonstrates a 3D-printed proof-of-concept self-controlling CVT (Continuously Variable Transmission). The device uses centrifugal force acting on weighted arms — similar in principle to a steam governor — to automatically adjust its gear ratio in response to input speed or output load, without any manual intervention.
