## Overview

Robert Murray-Smith demonstrates the destructive distillation of HDPE plastic into a waxy fuel product, then pivots to discuss a recent paper from Rensselaer Polytechnic Institute showing how that same waxy substance can be converted into spider silk-like protein via bacterial fermentation. The key takeaway is that bacteria (*Pseudomonas putida*) can accomplish what no chemist can — transforming polyethylene into a biodegradable, high-strength bioplastic — offering a genuinely useful end-of-life pathway for waste plastic beyond simply burning it.
