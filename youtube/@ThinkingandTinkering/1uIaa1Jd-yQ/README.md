## Overview

Robert Murray-Smith demonstrates how to make a fully biodegradable, eco-friendly fabric ink using only natural, food-safe ingredients. The video walks through a stepwise process of building an ink from a carrier base, binder, and coloring agent, culminating in an ink suitable for brush, pen, or screen printing on cloth. The finished ink is flexible, waterproof when dry, washable, and contains no harmful additives.
