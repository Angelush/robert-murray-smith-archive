## Overview

Robert Murray-Smith demonstrates how a cheap hand winch (£15) can be repurposed as a functional gear set for motors, generators, wind turbines, and other drives. He shows how to modify it by adding bearings and an output shaft, then connects a small DC motor with a speed controller to prove the concept. The key takeaway is that usable gear reduction can be achieved on a tight budget without complex or expensive components.
