## Overview
In this video, Luke (a contributor on Robert Murray-Smith's channel) builds a compact, foldable spray booth for indoor spray painting when outdoor conditions are unsuitable. The booth is constructed from PVC sheeting reinforced with builder's board and connected with cloth/tape hinges, making it collapsible for easy storage. The finished booth includes an extractor fan and a hanging hook, and is demonstrated working effectively for painting small items.

## Key Topics
- Motivation: indoor spray painting solution for bad weather
- Material selection for a lightweight, collapsible structure
- Frame reinforcement using butt joints
- Creating cloth hinges and tape hinges for foldability
- Adding an extractor fan for ventilation
- Adding a hanging hook for items being sprayed
- Demonstration of the completed booth in use

## Materials and Chemicals Mentioned
- **Stormlight** — thin, flexible PVC sheeting used as the main panel material (UK name; noted as possibly called something different elsewhere)
- **Builder's board (Citra Board in the US)** — rigid sheet material used to create a reinforcing frame around the PVC panels
- **Glue** — used to bond the builder's board frame to the PVC panels
- **Tape** — used to create the hinge joints between panels
- **Command hook** — used as a hanging point inside the booth (modified by melting and bending the tip)
- **Spray paint** — used in the demonstration (specific product not named)
- **Battery pack** — used to power the extractor fan

## Techniques and Methods
- **Butt joints** — simple frame construction technique to attach builder's board reinforcement strips
- **Cloth/tape hinges** — panels joined with tape on both sides to create flexible, foldable connections
- **Hole cutting** — cutting an aperture in one panel to seat the extractor fan
- **Heat forming** — melting and bending the command hook to improve its grip
- **Hanging and rotating** — suspending the workpiece on string so it can be flipped to spray both sides
