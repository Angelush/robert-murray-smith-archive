## Overview

In this video, Robert Murray-Smith follows up his previous portable rocket stove build (video 1403) by constructing a J-tube rocket stove core — the foundational component of a rocket mass heater. He cuts, welds, and tests a square-section J-tube, explaining key design ratios and airflow control, with the longer-term goal of enclosing it in a barrel to create a full mass heater.
