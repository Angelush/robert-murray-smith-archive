## Overview

Robert Murray-Smith explores the history of the pantograph and its refined successor, the eidograph — a precision drawing instrument invented by mathematician William Wallace in the 1800s. He designs and 3D prints a fully functional eidograph from scratch using only primitive shapes, demonstrating how it can enlarge, reduce, or copy drawings by adjusting pivot positions along its arms.
