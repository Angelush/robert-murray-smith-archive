## Overview

Robert Murray-Smith walks through how to approach choosing free CAD software, emphasising that most advice is personal rather than universal. He frames the decision around skill level, project requirements, and personal goals, offering a tiered set of recommendations from beginner to advanced. His main takeaway is to start simply, build confidence, and progress naturally rather than forcing yourself through an unnecessarily steep learning curve.
