## Overview

Robert Murray-Smith introduces the fundamentals of ink-making, demonstrating how to create a simple oil-based printing ink using recycled laser toner and boiled linseed oil. He then broadens the scope to explain "functional inks" — inks containing active materials like graphite, iron oxide, or cadmium sulfate that can serve purposes beyond decoration, enabling printed batteries, solar cells, supercapacitors, and thermoelectric generators.
