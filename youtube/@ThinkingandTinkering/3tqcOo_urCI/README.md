## Overview

This video, presented by Luke (a contributor on Robert Murray-Smith's channel), demonstrates how to make a ring from a coin using basic workshop tools. The process involves hammering the coin's edge to fold and compress it into a band, then drilling and filing out the center to create a wearable ring. The result is a piece of unique handmade jewelry suitable as a last-minute gift.
