## Overview

Robert Murray-Smith demonstrates a second method for making "UltraSand" — graphite/graphene-coated sand for water filtration — using bitumen (asphalt) dissolved in toluene as the carbon source. The method is simpler and significantly cheaper to run than the sugar-based method because it requires only 3 hours at 400°C rather than 7 hours at 750°C. The resulting material is effective at removing organic contaminants, pathogens, pesticides, colorants, and some heavy metals such as arsenic.
