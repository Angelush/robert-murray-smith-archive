## Overview

Robert Murray-Smith presents an experimental version of his Flynn motor replication, built specifically to allow flexible configuration testing. The device features swappable coil connections via brass pin terminals, enabling him to explore different circuit configurations that he found confusingly inconsistent across online sources. The video is primarily a build showcase, with plans to drive the switching coils from a computer-generated function generator signal fed through an amplifier.
