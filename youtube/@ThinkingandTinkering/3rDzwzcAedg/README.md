## Overview

Robert Murray-Smith demonstrates how to create "found art" — a deer's head sculpture — using salvaged parts from a child's bicycle (the seat and handlebars). The piece is inspired by Picasso's famous *Bull's Head* sculpture, and the video doubles as a broader meditation on creativity: that it is a transferable, trainable skill best exercised by staying observant of what's around you rather than seeking specific materials.
