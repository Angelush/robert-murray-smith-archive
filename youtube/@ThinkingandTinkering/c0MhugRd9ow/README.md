Robert Murray-Smith completes and test-rides a homemade electric quad bike built primarily from bamboo, demonstrating that bamboo is robust enough to serve as a structural material in a vehicle chassis. The final version incorporates an improved gear ratio using a bicycle cassette, solving earlier overheating and performance issues. The project proves bamboo's viability for demanding mechanical applications.
