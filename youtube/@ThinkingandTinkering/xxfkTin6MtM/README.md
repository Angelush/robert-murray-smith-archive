## Overview

This is a competition entry video in which the creator attempts to build a homemade supercapacitor using cheap, household materials. Three cells are assembled from carbon-coated felt paper, aluminum foil current collectors, and a vinegar-salt electrolyte, then connected in series. After several charge/discharge cycles at 6V, the device successfully lights an LED, though the creator notes the aluminum corroded and acknowledges limited understanding of the underlying science.
