## Overview

This short tips-and-tricks video, presented by Luke on the Robert Murray-Smith channel, demonstrates how to use WD-40 to remove super glue from timing belts and other surfaces. The method — spraying WD-40, allowing it to soak, then scraping — proves far more effective than acetone alone. The takeaway is that WD-40's degreasing properties loosen cyanoacrylate adhesive quickly and cleanly.
