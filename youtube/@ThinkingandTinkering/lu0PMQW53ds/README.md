## Overview

Robert Murray-Smith addresses the frequently asked question of how capacitors, supercapacitors, and batteries differ, arguing that these are not rigid, mutually exclusive categories but points on a continuum. He traces the structural and electrochemical transitions from simple parallel-plate capacitors through electrolytic capacitors and supercapacitors to lithium-ion batteries, showing how the boundaries blur at each step. His key takeaway is that how you define and distinguish these devices depends heavily on the functional context in which you need to use them.
