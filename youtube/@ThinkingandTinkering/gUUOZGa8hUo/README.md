## Overview

Robert Murray-Smith demonstrates a practical method for improving earth battery performance by isolating individual electrode cells inside short sections of plastic pipe driven into the ground. The core problem — that connecting earth batteries in series normally fails because the shared earth electrolyte shorts them out — is solved by physically separating each cell, allowing series voltage addition. Using four isolated carbon/lead cells, he achieves ~0.94V in series compared to ~0.3V from a single cell.
