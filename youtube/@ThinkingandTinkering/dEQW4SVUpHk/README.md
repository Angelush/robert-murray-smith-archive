## Overview

Robert Murray-Smith demonstrates an improved method for constructing an Archimedes screw using recycled CDs as the helical vanes. The key discovery is that enlarging the central hole of a CD from 15mm to 48mm dramatically increases the disc's flexibility, allowing it to conform snugly to ABS pipe and be heat-welded into a functional helix. He tests the resulting screw as a potential hydro-generator, spinning it with a blower to validate the blade pitch.
