## Overview

Robert tests a vertical axis wind turbine (VAWT) rotor he built from a scrap tumble dryer drum, demonstrating that it spins in real outdoor conditions. After an initial indoor test with a hairdryer drew skepticism, he shows the rotor turning in a very light natural breeze (0.9–1.2 m/s), validating the concept. The video is a brief proof-of-concept update confirming the drag rotor design works in a real-world environment.
