## Overview

Robert Murray-Smith demonstrates how to build a DIY camera lucida — an optical drawing aid — by salvaging a prism from a broken LCD projector. The device superimposes a scene onto paper so the user can trace it accurately. He argues that discarded LCD projectors are a rich source of high-quality optics, and that making a camera lucida from one is a genuinely mind-blowing experience.
