## Overview

Robert Murray-Smith demonstrates how to build a low-friction, magnetically assisted bearing using steel knitting needles and basic craft materials. The bearing works by using opposing magnets to support a needle point in a small indented cradle, dramatically reducing friction. He tests it with a paper daisy wheel that spins freely with just a breath of air.
