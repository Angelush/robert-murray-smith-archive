## Overview

Robert Murray-Smith demonstrates that his graphene-based conductive ink ("our ink") can be painted onto paper in a spiral pattern to create a functional coreless transformer. By attaching the painted coil to a joule thief circuit generating an ~80–100 kHz AC signal, he shows that AC voltage is transferred inductively between two isolated painted spirals, while DC voltage does not pass through — confirming true transformer action rather than simple leakage.
