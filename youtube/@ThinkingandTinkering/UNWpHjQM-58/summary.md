## Overview
This short tutorial demonstrates how to use the free, open-source VLC media player to convert video files between different formats and codecs. The presenter encountered a real-time problem: an MKV file was unsupported by his video editing software, and VLC's built-in conversion tool resolved it instantly. The main takeaway is that VLC is a quick, no-cost solution for codec/format incompatibility issues.

## Key Topics
- Video codec incompatibility with video editing software
- Using VLC's Media > Convert/Save feature to transcode video files
- Converting MKV to MP4 with the H.264 codec
- Brief mention that VLC can also convert audio files

## Materials and Chemicals Mentioned
None mentioned.

## Techniques and Methods
- Opening VLC and navigating to **Media > Convert/Save**
- Adding a source file via the file browser
- Selecting the output format (MP4) and codec (H.264) from the profile dropdown
- Setting a destination path and filename, then starting the conversion
