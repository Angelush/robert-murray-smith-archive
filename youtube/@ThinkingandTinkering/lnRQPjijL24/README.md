## Overview

Luke (presenting on Robert Murray-Smith's channel) demonstrates how to build a contra-rotation pulley mechanism as part of the "mechanical alphabet" series. Using two pairs of MDF pulleys, a leather bootlace belt, and a single twisted belt configuration, the mechanism drives two output shafts spinning in opposite directions simultaneously. The build highlights how this same principle appears in Robert's Wimshurst machine video and in contra-rotating helicopter rotor systems.
