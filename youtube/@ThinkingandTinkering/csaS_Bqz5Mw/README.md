## Overview

Robert Murray-Smith demonstrates how to weld the electrode tabs on a homemade supercapacitor, following up on a previous battery-building video where viewers requested this detail. The device shown weighs 160 grams with six carbon plates and is expected to store around 20 watt-hours per kilogram. The process uses an inexpensive Chinese tab welder operated via a foot pedal, making the technique accessible to home experimenters.
