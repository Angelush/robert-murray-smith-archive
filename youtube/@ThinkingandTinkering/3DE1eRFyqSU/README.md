## Overview

Robert Murray-Smith tests whether the hemp-based bioplastic he developed in a previous video — made from hemp, casein, and tea — can be processed using injection moulding. Using a machine donated by Kings School Canterbury, he successfully injection moulds the material into a disc shape, marking a promising first result while acknowledging significant optimisation work remains.
