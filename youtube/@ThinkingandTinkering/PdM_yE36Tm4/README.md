## Overview

Robert Murray-Smith demonstrates how to build a DIY light box from a spare 600×600mm LED lighting panel, showing it is far cheaper and simpler than buying a commercial equivalent. Using builders board (Cintra board) for the housing and salvaged hardware, he produces a finished, gift-ready light box for around £14–20, compared to £80–100 for a store-bought version. The project is presented as an ideal handmade gift for artists, designers, or anyone who does tracing work.
