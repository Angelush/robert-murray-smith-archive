## Overview

Robert Murray-Smith demonstrates how to build a low-cost DIY centrifuge from acrylic discs and a hobby drill, motivated by the need to separate metal nanoparticles from solution without spending £200 on a commercial unit. The build uses heat-formed acrylic, a hole saw, and a bench-mounted hobby drill as the spin motor, achieving speeds up to 12,000 RPM. The total cost is around £10, making it accessible to cash-strapped independent researchers.
