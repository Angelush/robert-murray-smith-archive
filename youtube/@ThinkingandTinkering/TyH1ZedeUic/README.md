## Overview

Robert 3D prints both a toroidal turbine blade and a standard three-bladed wind turbine blade, then tests them under identical conditions (same swept diameter, same fan, same distance) to see which generates more voltage. The result is that the toroidal blade performs no better than the standard blade, which Robert explains is unsurprising given that both are governed by the same physical limit (Betz's Law).
