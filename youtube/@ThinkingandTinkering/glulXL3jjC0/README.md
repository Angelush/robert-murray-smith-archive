## Overview

A beginner-friendly walkthrough for getting your first print off the Qidi X Max 3 3D printer. Robert covers the full pipeline from finding a model online, through slicing it with the included software (a customised build of PrusaSlicer), to transferring the G-code to the printer via USB and starting the print. The main takeaway is that the factory presets provided by Qidi are good enough for the vast majority of first prints, so beginners can keep it simple.
