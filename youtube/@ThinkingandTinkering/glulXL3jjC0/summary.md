## Overview
A beginner-friendly walkthrough for getting your first print off the Qidi X Max 3 3D printer. Robert covers the full pipeline from finding a model online, through slicing it with the included software (a customised build of PrusaSlicer), to transferring the G-code to the printer via USB and starting the print. The main takeaway is that the factory presets provided by Qidi are good enough for the vast majority of first prints, so beginners can keep it simple.

## Key Topics
- Where to find free 3D models to print (Thingiverse, Cults 3D, Printables, etc.)
- What slicing is and why it is necessary
- Overview of common slicer programs: Cura and PrusaSlicer
- Installing the Qidi-supplied slicer from the included USB stick
- Importing STL files exported from Tinkercad
- Basic object manipulation in the slicer: move, scale, rotate, place-to-face, arrange
- Key print settings: layer height, material profile, supports, infill, and brim
- Slicing to G-code and previewing the result layer by layer
- Exporting G-code to a USB stick and starting the print on the machine

## Materials and Chemicals Mentioned
- **PLA** — described as the most popular and easiest-to-use filament material; Robert's default choice
- **ABS** — listed as an available material profile in the slicer
- **PG** — listed as an available material profile
- **TPU** — listed as an available material profile (flexible filament)

## Techniques and Methods
- **Slicing** — converting a 3D model into layers of G-code that the printer executes
- **Infill patterning** — printing a sparse internal structure (default 25%) rather than a solid object to save material
- **Brim** — printing an extra perimeter around the first layer to improve bed adhesion, used here for a thin spiral object
- **Support structures** — discussed conceptually for overhanging geometry; not used in this print as the objects are flat
- **USB transfer** — exporting G-code to a USB stick and inserting it into the printer to print without a PC connection
