## Overview

Robert Murray-Smith demonstrates how to repurpose the ceramic ring magnets found inside old speakers to build a functional brushless DC motor. He shows the disassembly process, fabrication of the rotor using steel angle brackets to redirect the magnetic flux into discrete north/south poles, and a working proof-of-concept test using hand-wound coils in a Y configuration driven by an electronic speed controller.
