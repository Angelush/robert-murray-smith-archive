## Overview

In this video, Robert Murray-Smith demonstrates how to generate electricity from the endothermic reaction of urea dissolving in water, using a Peltier (thermoelectric) device salvaged from a mini fridge. By chilling one side of the Peltier with urea solution while the other side absorbs ambient heat, he creates a temperature gradient sufficient to run a small motor. The video is a companion to video 1268, which used the hot side (quicklime + water, exothermic) instead.
