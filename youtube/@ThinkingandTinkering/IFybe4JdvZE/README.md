## Overview

This video, presented by Luke (a collaborator on the channel), documents a first-time experience using a MIG welder after having prior experience with MMA (arc) welding. Luke walks through the setup process — feeding wire, configuring for gasless operation — then performs several practice welds, progressively dialling in settings to achieve a clean "stacking coins" bead. The key takeaway is that MIG welding is well-suited to thin sheet metal work that arc welding struggles with.
