Luke completes a tabletop arcade cabinet build, walking through the final assembly steps including button and joystick installation, RetroPie firmware setup on a Raspberry Pi 2, wiring to USB encoders, custom printed side graphics, and a protective acrylic screen panel. The finished machine runs classic arcade games like Street Fighter and Mortal Kombat and is presented as a fully functional retro arcade cabinet built from scratch.
