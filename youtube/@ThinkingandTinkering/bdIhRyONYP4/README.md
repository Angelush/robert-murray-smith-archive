## Overview

Robert Murray-Smith demonstrates how to make waterproof cloth using casein (milk protein) dissolved with borax and plasticized with glycerin, then fixed onto cotton fabric using tannic acid from tea. The casein coating, once treated with tannin, becomes rubbery and water-repellent, causing water to bead off rather than soak in. Robert frames this as part of a broader argument for using dairy biopolymers as sustainable alternatives to conventional plastics.
