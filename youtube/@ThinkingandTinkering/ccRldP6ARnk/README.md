## Overview

A hands-on tutorial video showing how to construct the body armature for a papier mache Madonna figure. Robert builds up the figure using newspaper cones, masking tape, and a central rod, working from a cast doll's head created in Part 1, and ends just as the paper mache covering stage is about to begin.
