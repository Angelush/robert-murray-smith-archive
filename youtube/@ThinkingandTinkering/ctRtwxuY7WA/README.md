## Overview

Robert Murray-Smith demonstrates how a complete beginner can get started with 3D printing by combining freely available files from repositories like Thingiverse with simple modifications in Tinkercad. Using himself as the subject, he creates a Doctor Who-themed bobblehead by merging a 3D scan of his own head with a downloaded Doctor Who figurine body. The key takeaway is that you don't need to master 3D modelling from scratch — remixing existing open-source designs is a legitimate and community-approved way to create personalised, satisfying objects.
