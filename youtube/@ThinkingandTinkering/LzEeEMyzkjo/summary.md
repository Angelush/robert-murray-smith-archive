## Overview
Robert Murray-Smith demonstrates how to fabricate a homemade string anchor (tail-piece/tuning head) for a ukulele using a repurposed electrical terminal block and a brass bar. The video is part of a series on building a complete homemade ukulele from found parts. He improves on a design he found online, producing a neater, more functional string-clamping mechanism.

## Key Topics
- Design concept for a headless-style ukulele string anchor
- Modifying electrical terminal blocks for use as string holders
- Soldering brass components together with proper technique
- Spacing calculations for ukulele string layout
- Integration of the anchor with the ukulele neck/nut

## Materials and Chemicals Mentioned
- **Electrical terminal block** (5-amp and 3-amp) — used as the base for string-clamping inserts
- **Brass bar** — 1 cm wide, ~2.3 mm thick, cut to 3.5 cm for the ukulele-sized anchor
- **28-gauge magnet wire** — used to hold brass inserts in place during soldering
- **Flux** — applied to brass bar before soldering
- **Solder** — used to permanently fix the terminal block inserts to the brass bar

## Techniques and Methods
- Splitting terminal block inserts free using a trimming knife
- Filing/flattening the rounded bottom of brass inserts in a vice for stability
- Temporarily securing parts with wire before soldering to prevent sliding during heat
- Soldering components while working on a longer stock piece for easier handling, then cutting to final size
- Drilling fixing holes in the finished brass assembly
- Spacing strings evenly (3 mm margins, 9 mm center-to-center)
