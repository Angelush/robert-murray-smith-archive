## Overview

Robert Murray-Smith demonstrates how to fabricate a homemade string anchor (tail-piece/tuning head) for a ukulele using a repurposed electrical terminal block and a brass bar. The video is part of a series on building a complete homemade ukulele from found parts. He improves on a design he found online, producing a neater, more functional string-clamping mechanism.
