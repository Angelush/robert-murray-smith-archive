## Overview

Robert demonstrates a low-cost, off-the-shelf home heating solution assembled from two inexpensive items purchased on Amazon for a combined £29. The system pairs a Chinese paraffin/kerosene stove with an "Outdoor Cozy" heat concentrator attachment, and the key insight is that running it on homemade biodiesel (from a previous video) reduces fuel cost to roughly 50p per litre versus £3–4 for commercial kerosene.
