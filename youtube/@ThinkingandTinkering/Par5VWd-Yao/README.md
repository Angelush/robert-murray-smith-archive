## Overview

Robert Murray-Smith demonstrates how to make traditional iron gall ink (ferrogallic ink) from scratch, using oak galls collected from nature, ferrous sulfate, and gum arabic. He explains the underlying chemistry that makes this ~1400-year-old ink archival quality, and closes by noting that making and selling this ink is a viable commercial opportunity.
