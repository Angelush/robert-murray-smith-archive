## Overview

Robert Murray-Smith demonstrates how to build a small wind-powered generator using a bicycle bottle dynamo, a magnetic gearbox he developed in a previous video, and propeller blades cut from plastic drainage pipe. The system converts slow propeller rotation (~60 RPM) into the higher speeds needed to generate useful electricity, suitable for charging phones or powering lights while camping or cycling.
