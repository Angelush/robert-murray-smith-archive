## Overview

Robert demonstrates several wind turbine blade designs mounted on a homemade DVD-drive-based generator, testing each with a hair dryer and measuring milliamp output. The video builds on a previous project (converting a DVD drive motor into a generator) and explores both conventional and unconventional rotor configurations. The standout finding is a flat disc impeller design inspired by inventor Mike Waters, which Robert argues could be cheaply deployed in large numbers on flat building surfaces.
