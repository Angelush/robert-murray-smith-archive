## Overview

This video is an announcement by Jackie Marie Smith (Rob's wife) and Robert Murray-Smith for a live interactive YouTube event titled "Make A Battery With Rob," scheduled for 27 October 2018. Participants would work along in real time using a materials kit to fabricate a small battery at home. The event was conceived as a solution to the problem of people being unable to attend FWG's in-person workshops, making the guided experimentation experience globally accessible.
