## Overview

Robert Murray-Smith demonstrates how to make "UltraSand" — graphene-coated sand — using only ordinary builder's sand and table sugar. The material is effective for water purification, removing pathogens, organic substances, and heavy metals like arsenic. The process replicates a published academic paper and is presented as accessible to home experimenters with basic equipment.
