## Overview

This video demonstrates how to build a simple DIY homopolar motor using just three common items: a battery, a magnet, and copper wire. The presenter (Luke) explains the basic working principle of a homopolar motor and shows two designs, with the preferred version using a copper wire coil wound around a broom handle for shape. The result is a continuously spinning wire assembly driven purely by electromagnetic interaction.
