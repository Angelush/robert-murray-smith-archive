## Overview

Robert Murray-Smith demonstrates how to convert a cheap drone motor into a working speaker using a piece of cellophane as a vibration membrane. The build is extremely simple — the motor acts as an inductor/electromagnet that vibrates the cellophane diaphragm when connected directly to an amplifier output. The key insight is that any inductor (motor, ferrite, air core coil) can produce sound this way, making it essentially a DIY "sound bug."
