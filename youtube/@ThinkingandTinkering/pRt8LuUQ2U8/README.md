## Overview

This video, presented by Franz as part of a challenge to build a useful machine from a bicycle, demonstrates how a stationary exercise bike can be converted into an electricity generator. Three different generator setups are tested — a bicycle dynamo, a car power window motor, and a windshield wiper motor — each capable of powering or charging various household devices. The main takeaway is that with minimal components (a DC motor, a diode, and a battery), anyone can build a practical energy harvester from a bicycle.
