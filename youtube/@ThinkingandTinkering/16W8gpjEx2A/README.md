## Overview

Robert Murray-Smith guides Luke through a hands-on exploration of magnetohydrodynamic (MHD) electricity generation, starting from the basic principles of electromagnetic induction using solid conductors and magnets, then extending the concept to conductive fluids — salt water, and ultimately a metal-seeded flame. The key takeaway is that anything conductive moving through a magnetic field will generate electricity, which is the same principle underlying all motors, generators, and MHD systems.
