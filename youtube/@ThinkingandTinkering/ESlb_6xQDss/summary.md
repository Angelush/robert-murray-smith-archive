## Overview
A short practical video focused on two tips arising from Robert's ongoing Darwin wind turbine project. He demonstrates 3D printing a full turbine blade set in ABS and coating it with cyanoacrylate glue to compensate for unfavorable grain direction, then tests the blades using a salvaged air conditioning fan motor as a generator, achieving around 200 milliwatts from a 4 m/s fan.

## Key Topics
- Progress update on the Darwin wind turbine project
- 3D printing a full set of turbine blades in one print
- Using cyanoacrylate (crazy glue) to strengthen ABS prints with wrong-direction grain
- Salvaging a brushless DC motor from an air conditioning unit for use as a generator
- Baseline power output testing for later comparison inside the Darwin turbine structure

## Materials and Chemicals Mentioned
- **ABS filament** — used to 3D print the turbine blades
- **Cyanoacrylate (cyan acrylic / crazy glue)** — applied as a surface coat to bond and strengthen ABS with unfavorable print grain direction

## Techniques and Methods
- 3D printing large turbine blades in a single print run using a large-volume printer
- Surface coating ABS prints with cyanoacrylate glue to improve inter-layer strength
- Using a salvaged air conditioning fan motor as a wind generator
- Baseline power measurement using a multimeter and LED load at a controlled wind speed (4 m/s fan)
