## Overview

Robert Murray-Smith demonstrates how to build a touch-activated driftwood lamp using found objects collected from a local beach, inspired by the rocky island illustrations of Finnish artist Tove Jansson. The lamp uses a simple touch-sensor electronics kit combined with conductive ink and copper strip to allow the lamp to be switched on by touching anywhere on the driftwood base. The project emphasises upcycling and encourages viewers to repurpose scrap or found materials into functional objects.
