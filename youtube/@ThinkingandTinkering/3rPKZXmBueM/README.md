## Overview

Robert Murray-Smith demonstrates a working model of a Nu Vinci continuously variable transmission (CVT) built from 3D-printed parts and ping-pong balls. He explains how friction-based machines exploit the varying radius of a sphere to achieve a continuously variable gear ratio, and shows the finished model achieving a gear range from 1:1 to 3:1.
