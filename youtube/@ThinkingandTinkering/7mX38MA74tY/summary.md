## Overview
Robert announces changes to the channel's membership structure as the channel approaches 500,000 subscribers. He introduces three tiers — Basic, Members, and VIP — each offering increasing levels of engagement and perks, including a new Discord server and a dedicated members questions video playlist for VIP subscribers.

## Key Topics
- Channel growth milestone approaching 500k subscribers
- Challenges managing comments and replies solo (without Luke)
- Restructured membership tiers: Basic ($1.99/mo), Members ($4.99/mo), VIP ($8.99/mo)
- Basic tier: shoutouts, thanks, and priority comment replies
- Members tier: access to members-only archive videos (graphene, batteries, etc.)
- VIP tier: Discord server access and ability to request dedicated videos
- New "Members Questions" playlist exclusive to VIP members
- New Discord server launch (named "TnT")
