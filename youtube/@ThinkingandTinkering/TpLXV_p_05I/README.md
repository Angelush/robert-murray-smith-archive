## Overview

Robert Murray-Smith demonstrates how to create a touch-activated light switch by painting conductive ink onto any surface — paper, card, or even a wall. The switch works by connecting the painted pad to a cheap capacitive touch sensor module, allowing a lamp to be toggled on and off by touch. The key insight is that the switch can be placed anywhere paintable, in any shape, and still works through a protective covering.
