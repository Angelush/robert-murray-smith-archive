## Overview

Robert Murray-Smith builds the frame for a "Dynapod" — a 1970s-era pedal-powered multi-purpose power unit — using a stripped bicycle frame and salvaged materials. The project is inspired by the self-sufficiency movement of the 1970s and aims to create a "Swiss Army knife" of pedal power that can drive different mechanical attachments. The key insight motivating the design is that direct mechanical power transmission is roughly 70% more efficient than converting pedal power to electricity first.
