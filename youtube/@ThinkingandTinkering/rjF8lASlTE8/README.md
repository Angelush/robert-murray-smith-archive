## Overview

Robert Murray-Smith repurposes two children's bicycle wheels to build a simple gravity battery/generator, connecting them vertically with a polypropylene belt and attaching a hand-crank drill generator to the upper wheel. Dropping weights drives the upper wheel to generate electricity. The first attempt partially works — the LED panel briefly lights up — but reveals clear failure points (belt slipping, weight hitting the floor stopping generation) that will guide improvements in a follow-up video.
