## Overview

Robert Murray-Smith demonstrates how to build a cheap and easy DIY ethanol stove using a repurposed cast iron concrete test mold as the base, a tin can as the fuel reservoir, and an optional steel tube chimney to boost heat output. The main takeaway is that for around £25 total, anyone can create a functional room heater or decorative fire without tools, flues, or professional installation — a practical solution in the context of UK energy costs and potential blackouts.
