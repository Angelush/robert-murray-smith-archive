## Overview

Robert Murray-Smith demonstrates a working continuously variable transmission (CVT) built almost entirely from 3D-printed parts and rubber dog toys. Two solid rubber rings (toroids) sit in a printed cradle, and tilting a pair of balls between them changes the output speed ratio smoothly without changing input speed. He argues this mechanism has real-world utility — particularly for wind turbines — where maintaining a constant output speed regardless of variable input is essential.
