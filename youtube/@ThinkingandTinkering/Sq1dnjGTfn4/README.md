## Overview

Robert Murray-Smith explores propane and bio-propane as practical alternatives to petrol, beginning with a demonstration of how trivially simple it is to convert a petrol generator to run on LPG. He then traces the evolution of bio-propane production — from pyrolysis and fermentation of biomass to cutting-edge biological methods using algae and bacteria — concluding that these technologies make propane a genuinely renewable "drop-in" fuel that requires no changes to existing machinery.
