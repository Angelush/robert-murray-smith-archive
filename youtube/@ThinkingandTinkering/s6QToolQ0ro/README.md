## Overview

This video, presented by Luke on Robert Murray-Smith's channel, demonstrates how to salvage 18650 cylindrical and flat lithium-ion cells from donated laptop battery packs and rebuild them into functional 12V DIY battery packs. Out of roughly 83 extracted cells, about half were found to be still viable. The main takeaway is a practical recycling workflow: test salvaged cells, discard dead ones responsibly, and assemble working cells into reusable packs using nickel strip connections and a thermal cutout for basic safety.
