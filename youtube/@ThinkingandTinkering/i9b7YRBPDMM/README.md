## Overview

This short tutorial demonstrates the correct technique for applying silicone sealant, presented by a collaborator named Luke. The presenter shares a tip he learned after struggling with silicone application in a previous project: using washing-up liquid on a gloved finger to smooth the bead into a clean, professional seal. The key takeaway is that a small amount of dish soap prevents the silicone from sticking to your fingers and allows it to be pressed firmly into gaps.
