## Overview

Robert Murray-Smith builds the frame for a large electric kick scooter powered by a 16-inch hub motor, fabricated primarily from repurposed steel shelving and salvaged moped parts. This is Part 1 of the build series, covering the design, cutting, and welding of the main frame structure. Being only a few weeks into welding, Robert embraces an overbuilt approach, prioritising strength and practicality over perfection.
