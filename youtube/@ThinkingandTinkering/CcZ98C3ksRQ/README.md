## Overview

Robert Murray-Smith uses the analogy of restricted palettes in painting — as practiced by artists like Turner — to argue that having fewer, well-mastered tools produces better results than accumulating every tool available. He extends this philosophy across carpentry, chef knives, CAD software, and 3D printing, concluding that the single best "tool" anyone can have is simply the ability to get up and start doing.
