## Overview

Robert Murray-Smith explores shape memory materials — starting with the well-known metal alloy nitinol and revealing that ordinary PLA 3D printing filament also exhibits shape memory behaviour. He demonstrates PLA recovering its printed shape when dipped in warm water, then discusses the potential and limitations of nitinol-based heat engines, concluding that PLA's shape memory opens exciting possibilities for low-thermal-difference engines and novel 3D printing applications.
