## Overview

Robert Murray-Smith demonstrates how to build a functional rocket stove using only large-diameter plumbing fittings — no welding, no specialist tools, and no prior building experience required. The stove is assembled by hand-tightening standard pipe components and sealing the joints with high-temperature silicate putty, making it accessible to virtually anyone. The result is a working, aesthetically pleasing rocket stove suitable for outdoor cooking and heating.
