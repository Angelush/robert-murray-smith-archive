## Overview

Robert Murray-Smith demonstrates how a cordless electric drill can be converted into a hand-cranked generator, showing that both devices share the same four core components. He compares a commercial hand-crank charger (£20) with a DIY version built from a free old drill plus a £1 voltage regulation circuit board, concluding both produce comparable output (up to ~18 watts) at dramatically different costs.
