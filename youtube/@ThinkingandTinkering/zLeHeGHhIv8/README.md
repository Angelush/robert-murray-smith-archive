## Overview

Robert Murray-Smith demonstrates how he upcycled four old office desks salvaged from a lab remodel rather than throwing them away. Three desks were converted into storage cupboards, while the fourth was transformed into a coffee table using the desk pedestals as a base structure and desktop plywood for the top, legs, and shelf components. The video is a practical, no-frills joinery project emphasising resourcefulness and waste reduction.
