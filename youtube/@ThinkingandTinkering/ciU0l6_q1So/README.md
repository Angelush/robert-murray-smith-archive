## Overview

In this short build episode, Robert disassembles a belt sander to salvage its internal components for use in a Van de Graaff generator. He extracts the rollers, motor, bushes, stator, and rotor, then prepares the bottom roller assembly for use as the Van de Graaff's belt-drive roller, concluding with a nearly finished bottom roller ready to mount on the generator base.
