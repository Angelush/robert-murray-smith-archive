## Overview

In Part 3 of his Wimshurst machine build series, Robert Murray-Smith constructs the body and drive mechanism of his electrostatic generator. He assembles the frame using repurposed mahogany coffee table components, fabricates a twisted belt pulley drive system from leather boot laces, and brings the machine to a near-complete state — with only the collector combs and neutralizing bars remaining for the next video.
