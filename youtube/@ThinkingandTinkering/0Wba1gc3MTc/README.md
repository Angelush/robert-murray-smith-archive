## Overview

Robert Murray-Smith explores the challenge of self-starting lift-type vertical axis wind turbines (VAWTs), specifically the Darrieus design, which can exceed wind speed but cannot start on its own. He presents a 3D-printed prototype designed in Tinkercad that uses a 120-degree twisted airfoil geometry to overcome this limitation. The video demonstrates the turbine being assembled and tested with a modular generator system from a previous video.
