## Overview

Robert Murray-Smith explores the concept of a variable stroke piston, inspired by the Miller engine used in the Mazda Millennia production car. He designs and 3D prints a working mechanical model that demonstrates how varying the eccentric position of a piston's internal slide changes the stroke length — from zero movement to maximum throw — with applications in variable compression engines, variable pumps, and CVTs.
