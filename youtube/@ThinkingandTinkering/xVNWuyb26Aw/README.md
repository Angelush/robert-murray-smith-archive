## Overview

This video demonstrates how to make bioplastic guitar picks using a simple five-ingredient recipe based on casein (milk protein). The presenter walks through the full process from mixing chemicals to pressing a mold, cutting picks, and finally playing guitar with the finished product. The result is a functional, solid pick that the presenter plans to use permanently.
