## Overview

Robert Murray-Smith explores three fundamental gear types — spur, helical, and double helical (herringbone) — comparing their properties, trade-offs, and fabrication methods. He demonstrates a 3D-printed display model combining all three gear types in a planetary-bearing arrangement, emphasizing that 3D printing democratizes access to complex gear geometries like herringbone gears that are prohibitively expensive to machine. The key takeaway is that herringbone gears in particular are underexplored and deserve much more experimentation now that 3D printing removes the cost barrier.
