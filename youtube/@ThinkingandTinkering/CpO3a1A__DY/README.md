## Overview

Robert Murray-Smith theorises the operating mechanism behind two devices he previously built: a "perpetual battery" and a "wave generator." He argues both are driven by triboelectric-style charge separation that occurs when an ion-carrying fluid moves across a surface, and that extracting usable current requires a resistance imbalance between the two electrodes — achievable either by using different materials or by changing electrode geometry. He views the current prototypes (graphite-in-wax clay rods) as primitive proof-of-concept, with significant room for improvement using better carbon-loaded materials.
