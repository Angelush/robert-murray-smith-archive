## Overview

This video demonstrates how to make a simple slow-burning fuse using only three readily available ingredients: potassium nitrate, sugar, and absorbent string. The presenter makes the fuse to ignite a model rocket and tests two different string types, finding both burn approximately 5 cm per minute.
