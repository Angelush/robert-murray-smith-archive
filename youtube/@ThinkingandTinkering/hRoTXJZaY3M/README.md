## Overview

This video is a progress update on an ongoing project to strengthen concrete using graphene, reporting results from over 400 experiments. The team demonstrates compressive strength measurements using a rebound (Schmidt) hammer, showing a graphene-enhanced block reaching 52 N/mm² — roughly double the standard 26–28 N/mm². A key highlight is a "real-world simulation" block made under deliberately poor, informal conditions that still matched or exceeded standard concrete performance.
