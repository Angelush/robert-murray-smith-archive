## Overview

Robert Murray-Smith walks through the construction of a basic laboratory kiln suitable for temperatures up to 1200°C, explaining the three core components: the kiln body, the electronics, and the outer enclosure. The video focuses on building the brick-and-ceramic-blanket body assembly, emphasising that using properly rated commercial materials (rather than DIY mixes like perlite or vermiculite) eliminates long drying times and guesswork. The main takeaway is that a functional, properly-rated lab kiln can be built cheaply and easily using fire bricks, ceramic fiber blanket, and high-temperature silicone putty.
