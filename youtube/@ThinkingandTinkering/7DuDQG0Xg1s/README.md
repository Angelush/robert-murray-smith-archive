## Overview

Robert Murray-Smith demonstrates an upgraded version of his "exploding water" experiment, using five 2kV capacitors wired in parallel and a dedicated spark gap instead of a spark plug. The surprising finding is that the water ignites brilliantly on the polycarbonate board surrounding the spark gap, not inside the gap itself — a result he finds deeply puzzling and invites viewers to explain.
