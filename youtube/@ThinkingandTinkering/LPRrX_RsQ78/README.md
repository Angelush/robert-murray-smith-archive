## Overview

Robert Murray-Smith begins building an e-scooter frame from bamboo, using a salvaged drivetrain (motor and belt drive) from a discarded kids' e-scooter. He demonstrates his signature "stitch and glue" bamboo construction technique — drilling, slotting, binding with twine, and applying resin — to create a lightweight, low-cost frame. This is Part 1, ending with a finished structural frame ready for motor, battery box, and handlebars in the next video.
