## Overview

Robert revisits his Elegoo Mars 2 Pro resin printer to share an honest update on his ongoing attempts to create conductive 3D-printing resin by adding graphene — experiments that have so far failed. As a pivot, he demonstrates successfully printing a functional Fresnel lens using a plant-based translucent resin, proving that resin printers can produce working optical components.
