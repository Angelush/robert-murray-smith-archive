## Overview

Robert Murray-Smith replicates an axial magnetic gear design originally demonstrated by William Fraser, building it from simple materials (builders board, bolts, and permanent magnets). He demonstrates that two magnetic discs with differing pole pairs create a contactless gear reduction, and has a key insight: by fixing one disc and using electromagnets to create a rotating magnetic field, the design could form a novel hub motor.
