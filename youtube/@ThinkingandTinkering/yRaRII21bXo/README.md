## Overview

Robert demonstrates a range of metal-air batteries built on the graphene ink air cathode he developed in a previous video. Starting from a magnesium-air baseline, he swaps in zinc, aluminium, and iron (steel wool) as the anode material, using the same graphene-coated carbon cloth cathode and sodium chloride electrolyte throughout. The core takeaway is that the graphene cathode design is highly adaptable — experimenting with different metals and electrolytes is encouraged, and simple, locally available materials are sufficient to get started.
