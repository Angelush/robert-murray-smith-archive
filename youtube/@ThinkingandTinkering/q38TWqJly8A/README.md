## Overview

This video documents the construction of a DIY bandsaw nicknamed "Cutie," built primarily from a repurposed 16-inch wheel bicycle combined with salvaged electrical components. The creator walks through the machine's features, the build process, and demonstrates it cutting various materials. The main takeaway is that a functional and adjustable bandsaw can be fabricated from inexpensive recycled parts for as little as $12 for the donor bicycle.
