## Overview

Robert builds a DIY magnetiser/demagnetiser using salvaged squirrel cage rotors from induction motors found in a car park, repurposed into a complete unit housed in a found frying pan. The project pairs the coil with a bridge rectifier and a repurposed drill-case momentary safety switch, creating a practical tool for magnetising or demagnetising small items like screwdrivers. The build is intentionally constructed from scrap and leftover parts.
