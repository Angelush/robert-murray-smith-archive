## Overview

Robert Murray-Smith demonstrates how to build a bio-photovoltaic (biophotovoltaic) solar cell using moss and simple conductive materials. He explains that it is essentially a microbial fuel cell driven by photosynthesis — the microorganisms at plant roots produce electrons — and that the system is carbon-net-neutral because plants consume the CO₂ produced. The build is intentionally simple, aimed at science fair projects and home experimentation, with output best captured via a supercapacitor.
