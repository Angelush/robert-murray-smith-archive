## Overview

Robert Murray-Smith provides a practical guide to designing a wind turbine from first principles, arguing that most people approach the problem backwards by starting with a motor. His core advice is to begin with your local wind speed data, use that to determine what size turbine you can realistically build, and only then select a motor that matches the turbine's output torque and RPM.
